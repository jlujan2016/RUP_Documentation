Directriz: Integración de sistemas para aplicaciones J2EE
Esta directriz trata la integración de sistemas para una aplicación J2EE.
Relaciones
Elementos relacionados
  • Integrar el sistema
Descripción principal

La integración de sistemas para una aplicación J2EE supone el ensamblar módulos J2EE individuales que representan los subsistemas de implementación en una aplicación J2EE. El archivador resultante representa la compilación del sistema. Consulte la sección Directriz: compilaciones para aplicaciones J2EE para obtener más información sobre las compilaciones para aplicaciones J2EE.

El ensamblaje de módulos J2EE de subsistemas de implementación individuales puede requerir algún tipo de reconciliación entre estos módulos. Además, los resultados de la integración se recogen en la aplicación J2EE resultante y su descriptor de despliegue. Consulte la sección Directriz: ensamblaje de aplicaciones J2EE para obtener más información sobre la forma en que se ensamblan aplicaciones J2EE.

El archivador resultante y su descriptor de despliegue después sirven como entrada a las pruebas de integración del sistema.
