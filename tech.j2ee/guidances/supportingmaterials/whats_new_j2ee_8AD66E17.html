De 4.0 a 4.0.1

En este release se incluyen sólo actualizaciones editoriales.

De 3.2 a 4.0

El release 4.0 supone una reestructuración importante de este plug-in para adaptarlo en el entorno de Rational Method Composer. Todos los roles y productos de trabajo (artefactos) que existían en versiones anteriores de este plug-in se han eliminado. Se han sustituido por directrices para las tareas aplicables en RUP. Esto permite incluir el plug-in en la configuración de RUP clásico, sin confundir a los equipos que utilizan RUP clásico y no realizan desarrollo J2EE.

De 3.1 a 3.2

El release 3.2 es el resultado de adaptar el plug-in J2EE a J2EE 1.4. Puesto que los cambios principales de esta nueva versión de J2EE están relacionados con el soporte de servicios web, esto también se refleja en la versión actual del plug-in.

Cambios menores de formato.

De 2.0 a 3.0

El release 3.0 contiene relativamente pocos cambios. Los cambios consisten básicamente en el aspecto y apariencia y en cambios de estructura para acomodar los cambios al proceso RUP que subyace bajo ella.

Se han añadido nuevas guías de herramientas para nuevas actividades y para describir las nuevas características de despliegue en XDE.
