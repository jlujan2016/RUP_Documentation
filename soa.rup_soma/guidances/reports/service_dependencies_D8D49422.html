Informe: Dependencias de servicios
Este informe describe el modelo de servicio de forma exhaustiva, desde el punto de vista de las dependencias entre distintos servicios. Si utiliza paquetes, el documento muestra la estructura del modelo jerárquicamente. El informe se puede utilizar para describir todo el modelo de servicio en distintas etapas: durante la elaboración; mientras se observa la cartera de servicios existentes para identificar dependencias en los servicios que puede que utilice; y durante la construcción, cuando el diseño está completo, para informar sobre las dependencias en los servicios que haya diseñado. Este informe lo utilizan las distintas personas interesadas en el modelo de servicio, como es el caso de los arquitectos de software, los diseñadores de guiones de uso, los diseñadores, los verificadores, los revisores y los gestores.

Descripción principal

1. Breve descripción
Una introducción al modelo de servicio.

2. Listado de especificaciones de servicio
Esta sección presenta las especificaciones de servicio jerárquicamente, explica las dependencias entre ellas y muestra el contenido de cada paquete.

Si el modelo tiene varios niveles de paquetes, los que estén situados en la parte superior se presentarán primero. Los paquetes de éstos se presentarán a continuación, etc., así sucesivamente hasta los paquetes situados en la parte inferior de la jerarquía. Para cada servicio deberá proporcionar lo siguiente:

  • Su nombre.
  • Una breve descripción.
  • El nombre de su proveedor de servicios.
  • El nombre de la especificación de servicios que implementa.
  • El nombre de cualquier especificación de servicio que requiera.
  • El nombre de cualquier canal de servicio que se conecte a otros servicios, así como el nombre del servicio en el otro extremo del canal.
  • El nombre de cualquier partición de servicio en la que aparezca el servicio.

3. Diagramas del modelo de servicio
Los diagramas, principalmente diagramas de clase y diagramas de estructura compuesta, de todo el modelo de servicio se incluyen aquí.
