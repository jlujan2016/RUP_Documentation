Plantilla: Modelo de servicio en UML
Esta plantilla proporciona un documento de UML de compendio que se puede utilizar para capturar un modelo de servicio y todos sus detalles.

Descripción
Descripción principal

Esta plantilla se proporciona para IBM Rational Software Modeler v6.0.1 y los archivos adjuntos se componen de un modelo de plantilla reutilizable y también de la implementación correspondiente del perfil de UML para servicios de software (consulte Documentación técnica: Perfil de Lenguaje de modelado unificado (UML) para servicios de software). La plantilla se compone de un conjunto de paquetes que se corresponden con los distintos elementos de vista del modelo de servicio, éstos se pueden ver en la siguiente captura de pantalla.

El modelo de servicio de UML puede ser el artefacto de entrega principal, pero en muchos casos se entregará en conjunto con contenido adicional en formato de documento; en relación con esto, consulte Plantilla: Modelo de servicio en Word.
