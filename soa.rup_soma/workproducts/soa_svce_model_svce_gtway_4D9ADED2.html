Artefacto: Pasarela de servicio
Este artefacto se parece mucho a un elemento de modelo de servicio salvo que no representa un punto final desde el punto de vista de la implementación de una especificación de servicio porque sólo reenvía mensajes desde el límite de una partición de servicio a un servicio contenido dentro de la partición. De esta forma, permite que las particiones definan de forma estricta sus interfaces desde el punto de vista de las pasarelas de servicio.
Clases de producto de trabajo: Elemento de modelo
Objetivo

Las siguientes personas utilizan las pasarelas de servicio:

  • Arquitectos de software, para un entendimiento de la comunicación entre particiones.
  • Implementadores, para un entendimiento de los requisitos de mediación entre las particiones.
  • Aquellos que diseñan la siguiente versión del sistema, para entender la composición de las particiones y los servicios del modelo de servicio.
