Actividad: Modelado de servicio de objetivos
Modelado de servicio de objetivos le ayuda a descubrir servicios alineados empresarialmente y garantiza que no se pierdan los servicios importantes durante la descomposición de procesos o el Análisis de activos existentes. Mediante una clara articulación de objetivos empresariales, el Modelado de servicio de objetivos también proporciona un importante mecanismo para limitar la atención en las otras técnicas de identificación de servicio. Empieza con objetivos empresariales, los divide en subobjetivos y luego determina qué servicios son necesarios para cumplir los subobjetivos. Los KPI, medidas y sucesos asociados se identifican y hacen posible medir el rendimiento de los servicios identificados y compararlos con los subobjetivos.

Descripción

El Modelo de servicio de objetivos puede verse como una situación entre dos elaboraciones.  La primera elaboración captura la estructura inicial de los objetivos, subobjetivos, KPI y medidas para el proyecto, y correlaciona objetivos y subobjetivos con servicios candidatos siempre que sea posible.  Al final de esta elaboración puede haber objetivos o subobjetivos que todavía no se hayan correlacionado con un servicio. 

La segunda elaboración tiene lugar cuando se toman decisiones de exposición de servicios, momento en el cual se evalúan los servicios candidatos para alineación empresarial.  Cualquier servicio que se vaya a elegir para exposición debe correlacionarse con al menos un objetivo o subobjetivo del Modelo de servicio de objetivos.  Esto puede producir una actualización del Modelo de servicio de objetivos para documentar la correlación de servicios, y potencialmente añadir un nuevo objetivo o subobjetivo (sujetos a una cuidada evaluación para cumplir con los objetivos empresariales).

El Modelado de servicio de objetivos, similar a otras actividades de SOMA, se realiza normalmente con un enfoque que construye incrementalmente un Modelo de servicio de objetivos a través de diversas repeticiones.  Un área del Modelo de servicio de objetivos puede resultar semicompleta en el sentido descrito para la primera elaboración mientras que otra parte puede parecer completa desde el punto de vista descrito para la segunda elaboración.  Eventualmente, las repeticiones de Modelado de servicio de objetivos y Decisiones de exposición de servicio adicionales darán como resultado un Modelo de servicio de objetivos que se haya completado en el segundo nivel de elaboración para todas las áreas del modelo que son necesarias para completar el proyecto actual.

La información necesaria para crear una instancia del producto de trabajo Modelo de servicio de objetivos se puede reunir a través de entrevistas y sesiones de trabajo facilitadas.  Para ofrecer una estructura para las entrevistas o las sesiones de trabajo, se puede rellenar una plantilla del Modelo de servicio de objetivos durante las sesiones, ofreciendo a todos los participantes una infraestructura visible y en desarrollo de los objetivos y subobjetivos.
