Documentación: Directrices de estructura de modelo para RSx
Este documento presenta un conjunto de directrices sobre cómo estructurar modelos RSx (RSA, RSM y RSD).

Descripción principal

Resumen

Este documento proporciona un conjunto de directrices sobre cómo estructurar modelos RSA. Estas directrices resultan muy útiles al decidir sobre la estructura de los modelos.
