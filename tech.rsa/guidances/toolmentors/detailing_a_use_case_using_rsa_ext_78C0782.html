Descripción principal

Visión general

En esta guía de la herramienta se da por supuesto que el modelo de guión de uso se ha creado según las Directrices de estructurad de modelo para RSx. También se presupone que se ha creado un guión de uso siguiendo los pasos descritos en Encontrar actores y guiones de uso.

En esta guía de la herramienta se llevan a cabo los pasos siguientes:

  • Detallar el flujo de sucesos del guión de uso
  • Estructurar el flujo de suceso del guión de uso
  • Ilustrar relaciones con actores y otros guiones de uso <opcional>
  • Describir los requisitos especiales del guión de uso
  • Describir protocolos de comunicación
  • Describir condiciones previas del guión de uso <opcional>
  • Describir condiciones posteriores del guión de uso <opcional>
  • Describir puntos de ampliación <opcional>
  • Evaluar los resultados

Información adicional sobre la herramienta

Antes de detallar un guión de uso, debe decidir cómo desea capturar los detalles. Algunas opciones son:

  • Utilizar la ventana Documentación de modelo, que le permite documentar cualquier elemento de modelo seleccionado. Consulte Documentación de elementos de modelo.
  • Se puede crear un documento de especificación de guión de uso separado utilizando la plantilla que proporciona RUP (consulte las plantillas del Artefacto: Guión de uso). A continuación, el documento se puede asociar al elemento de modelo de guión de uso. Consulte Enlace de elementos de modelo con archivos externos.
  • También puede asociar los guiones de uso con requisitos y documentos mediante la integración de Rational RequisitePro. Para asociar un guión de uso a un requisito o documento de RequisitePro, consulte la ayuda de la integración de RequisitePro.  A continuación, se proporciona más información sobre la integración de RequisitePro.

Integración de Rational RequisitePro

Mediante la integración de Rational RequisitePro, puede editar los elementos del modelo de guión de uso desde sus requisitos asociados en las bases de datos y los documentos de RequisitePro. Puede asociar los guiones de uso con documentos de requisitos de RequisitePro, lo que permite desarrollar una definición textual del guión de uso en un documento RequisitePro de Microsoft Word. En los documentos de guión de uso de RequisitePro, puede elaborar el guión de uso con descripciones, flujos de sucesos, requisitos especiales y condiciones.

La integración también permite asignar la rastreabilidad de requisitos a elementos de modelo. Puede utilizar los requisitos asociados para rastrear las relaciones entre los elementos de modelo y otros requisitos mediante las capacidades de rastreabilidad de requisitos de RequisitePro. Si asocia los elementos de modelo con los requisitos, le permite revisar y valorar el impacto de los cambios de requisitos en los elementos de modelo.

Para obtener más información, consulte las directrices Gestión de guiones de uso mediante RSx y Rational RequisitePro.

Detallar el flujo de sucesos del guión de uso

  1. Abra el modelo de guión de uso.
  2. Navegue al guión de uso que desea detallar.
  3. Entre un flujo de sucesos detallado en la documentación asociada con el elemento de modelo.

Estructurar el flujo de sucesos del guión de uso

Puede ilustrar la estructura del flujo de sucesos con un diagrama de actividad. Para ello, siga los pasos que se indican a continuación:

  1. Pulse el botón derecho sobre el guión de uso en el Explorador de modelos, y añada un diagrama de actividad. Consulte Diagramas de actividad.
  2. (opcional) Añada divisiones.
  3. Añada el estado inicial, el estado final y las actividades. Consulte Nodos de control.
  4. Anote cada actividad con una descripción breve. Consulte Documentación de elementos de modelo.
  5. Conecte los estados de actividad con transiciones. Consulte Relaciones.
  6. Cree barras de sincronización (según sea necesario).
  7. Especifique condiciones de vigilancia en las transiciones (según sea necesario).

Ilustrar relaciones con actores y otros guiones de uso <opcional>

Este es un paso opcional que se lleva a cabo únicamente para aclarar la relación entre el guión de uso y sus actores y guiones de uso asociados. La adición de relaciones con actores se describe en Búsqueda de actores y guiones de uso. Las relaciones con otros guiones de uso se describen en Estructura del modelo de guión de uso.

Para obtener más información, consulte Modelado del comportamiento de un subsistema mediante diagramas de guiones de uso en la ayuda en línea.
