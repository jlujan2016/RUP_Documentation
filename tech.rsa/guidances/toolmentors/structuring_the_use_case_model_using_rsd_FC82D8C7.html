Descripción principal

Visión general

En esta guía de la herramienta, se presupone que ya se ha creado un modelo de guión de uso, rellenado con actores y guiones de uso, siguiendo los pasos descritos en Búsqueda de actores y guiones de uso.

Esta guía de la herramienta consta de los pasos siguientes:

  • Identificar requisitos comunes
  • Establecer relaciones de inclusión entre guiones de uso
  • Establecer relaciones de ampliación entre guiones de uso
  • Establecer generalizaciones entre guiones de uso
  • Establecer generalizaciones entre actores
  • Evaluar los resultados

Información adicional sobre la herramienta

Identificar requisitos comunes 

El compartimiento de guiones de uso comunes se consigue añadiendo relaciones entre guiones de uso existentes o creando nuevos guiones de uso. Para ello, siga los pasos que se indican a continuación:

  1. Abra el modelo de guión de uso.
  2. Navegue al paquete que contiene los guiones de uso. Consulte Directrices de estructura de modelo para RSx.
  3. En Búsqueda de actores y guiones de uso, se ha creado un diagrama de guión de uso que contiene todos los actores y guiones de uso. Navegue a dicho diagrama.
  4. Refactorice los guiones de uso y cree nuevos, según proceda. El procedimiento para crear nuevos guiones de uso se describe en Búsqueda de actores y guiones de uso. El detalle de los nuevos guiones de uso se describe en Detalle de un guión de uso.
  5. Añada relaciones entre los guiones de uso. En las secciones siguientes se describe cómo añadir cada tipo de relación.
  6. Documente cada relación. Consulte Documentación de elementos de modelo.

Cree diagramas de guión de uso adicionales y empaquételos según sea necesario para gestionar la complejidad. Consulte Adición de diagramas de guión de uso a modelos y Organización de elementos de modelo en paquetes.

Para obtener más información, consulte Modelado del comportamiento de un sistema mediante diagramas de guiones de uso.

Establecer generalizaciones entre actores

  1. Abra el modelo de guión de uso.
  2. Navegue al paquete que contiene los actores. Consulte Directrices de estructura de modelo para RSx.
  3. En Búsqueda de actores y guiones de uso, se ha creado un diagrama para capturar todos los actores. Abra dicho diagrama.
  4. Refactorice actores existentes y cree nuevos actores, según proceda. El procedimiento para crear actores se explica en Búsqueda de actores y guiones de uso.
  5. Añada generalizaciones entre actores. Consulte Relaciones de generalización.
  6. Si lo desea, documente cada relación de generalización. Consulte Documentación de elementos de modelo.

Para obtener más información, consulte Modelado del comportamiento de un sistema mediante diagramas de guiones de uso.

Evaluar los resultados

La herramienta puede producir informes que extraigan información de guión de uso en un formato fácil de revisar. También puede publicar un modelo entero en un formato adecuado para la visualización y la navegación en navegador web.

Para obtener más información, consulte Publicación de modelos y el tutorial Publicación de un modelo en un sitio web .
