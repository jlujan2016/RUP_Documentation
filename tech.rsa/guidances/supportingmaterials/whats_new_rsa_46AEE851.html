Novedades en RUP para RSx
Descripción principal

Si desea obtener una descripción del contenido de este plug-in, consulte Acerca de RUP para RSx.

A continuación se describen las nuevas características y cambios entre versiones.

  • De 3.0 a 3.1.
  • De 2.0.1 a 3.0.
  • De 2.0 a 2.0.1
  • De 1.0 a 2.0
  • 1.0

De 3.0 a 3.1

  • Cambios a las directrices de la estructura de modelo. 
  • Guía de herramientas RSx actualizada. 
  • Añadido el soporte de Rational Software Modeler (RSM).

De 2.0.1 a 3.0

  • Se ha refactorizado el contenido para que pueda ser reutilizado por las diferentes variantes RSA .
  • Se ha añadido el soporte para Rational Systems Developer (RSD). 

De 2.0 a 2.0.1

En este release se incluyen varias actualizaciones editoriales y otras correcciones menores.

De 1.0 a 2.0

  • Migración a las herramientas del compositor de métodos de Rational.
  • Ediciones menores (adición de descripciones breves, ortografía, idioma, accesibilidad, etc.).

1.0

Este es el release inicial de este plug-in.
