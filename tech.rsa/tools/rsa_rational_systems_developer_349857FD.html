Descripción principal
IBM® Rational® Systems Developer es un producto de diseño y construcción integrado creado a partir de Eclipse. Aprovecha todos los recursos de Eclipse e incluye plug-ins que permiten a los arquitectos de software y a los desarrolladores controlados por modelo crear aplicaciones C++ y Java ä J2SE con una buena arquitectura de modo que se aprovecha la versión 2 del lenguaje unificado de modelado (UML).
