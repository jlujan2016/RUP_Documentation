Navegación por el sitio web de métodos
En esta guía se explica cómo navegar por un sitio web de métodos típico.

Descripción principal

El sitio web del método consta de un conjunto de páginas HTML que puede ver con los navegadores Microsoft® Internet Explorer, Mozilla y Firefox.

Nota: algunas funciones, como Buscar y My View, implementadas como applets precisan JRE 1.4.2 o superior (puede bajar un JRE de http://java.sun.com/j2se). Como alternativa, puede publicar una configuración sin applets, que no tiene las funciones Buscar y My View, pero no precisa un JRE. Si el sitio que está visitando se publicó sin applets, entonces ni Buscar ni My View estarán presentes.

La figura siguiente muestra los principales elementos utilizados para examinar el sitio web del método.

Seleccione cualquier área de la figura si desea una breve explicación de cómo efectuar la navegación.

Elementos del entorno del navegador

Glosario

Si pulsa este botón se abrirá otra ventana que contiene el glosario con la lista alfabética de los términos utilizados en el sitio web del método, así como definiciones y enlaces con páginas de ejemplo.

Índice

Si pulsa el botón de Índice se abre otra ventana que contiene la lista alfabética de los temas del sitio web del método. Si pulsa un hiperenlace, la página relacionada aparece en la ventana principal.

Comentarios

Si pulsa el botón de Comentarios se abre otra ventana que contiene un mensaje de correo electrónico para comentarios de Rational, que hace referencia automáticamente a la página que aparece en la ventana principal.

Acerca de

Si pulsa el botón de Acerca de se abre una ventana emergente con el número de versión actual del copyright.

Buscar

La búsqueda permite especificar una palabra clave y buscarla en el proceso, de forma que todas las páginas que son relevantes para ese tema aparezcan en la ventana de resultados de la búsqueda. El programa de utilidad de búsqueda funciona con los temas de palabras clave, en vez de buscar cadenas de caracteres en las páginas de contenido del proceso.
Tenga en cuenta que el motor de búsqueda, implementado como applets, precisa JRE 1.4.2 o superior (puede bajar un JRE de http://java.sun.com/j2se). Si el sitio que está visitando se publicó sin applets, entonces Buscar no estará presente.

Imprimir

El botón Imprimir envía el contenido de la ventana principal a la impresora.

Marco de contenido principal

Este marco es en el que se muestra el contenido del método. El contenido está organizado en secciones que se pueden ampliar y contraer.  

Vistas

Una vista le permite navegar y utilizar un sitio web del método desde una perspectiva diferente. Estos son algunos ejemplos de vistas:

  1. La vista Equipo muestra todo el contenido del sitio web. Engloba todas las otras vistas y le permite navegar por el método desde distintas perspectivas (ciclo vital, proceso de entrega, roles, disciplinas, productos de trabajo, etc.). También proporciona información de iniciación.
  2. La vista Cómo empezar proporciona una visón de los conceptos básicos del método (principios clave, elementos fundamentales del proceso, arquitectura del método, fundamentos), información sobre la navegación del sitio web y mapas para el uso del sitio web desde la perspectiva de una determinada tecnología o parte interesada. También proporciona material de tipo apéndice, como las referencias y los recursos.
  3. La vista Conjuntos de roles le permite seleccionar el rol en el que está interesado, y ver todo el trabajo relacionado con ese rol. También puede ver disciplinas, dominios y guías de herramientas que son importantes para el trabajo de este rol.
  4. La vista Procesos de entrega le permite navegar por los ciclos vitales (como los ciclos vitales RUP clásico y modelado empresarial). El navegador de árbol contiene los elementos del ciclo vital del nivel de fase a los niveles de iteración, actividad y subactividad.  Se proporcionan tres estructuras de desglose para cada elemento del ciclo vital:
    • Estructura desglosada del trabajo que muestra las actividades y tareas
    • Asignación de roles que muestra los roles 
    • Uso de los productos de trabajo que muestra los productos de trabajo 

Tenga en cuenta que la estructura de desglose de trabajo puede tener un gráfico navegable que ilustre el trabajo que implica el elemento de ciclo vital asociado. Asimismo, observe que los descriptores de tareas de las estructuras de desglose de trabajo tienen pasos. Estos pasos se muestran como círculos con relleno de color azul. Un círculo azul vacío indica que el paso no se va a completar en este ciclo vital.
