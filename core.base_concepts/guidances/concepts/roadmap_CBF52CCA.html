Concepto: Mapa
Este tipo de guía resume un proceso, a menudo desde una perspectiva particular.
Descripción principal

Una instancia de mapa representa documentación importante para la actividad o el proceso con el que está relacionada. A menudo, actividades complejas como los procesos pueden entenderse mucho más fácilmente si se proporciona un ensayo con una hebra lineal de una creación de instancias de esta actividad.

Además de facilitar al profesional del proceso la comprensión de cómo se lleva a cabo el trabajo en el proceso, un mapa proporciona información adicional acerca de cómo las actividades y las tareas se relacionan entre sí a lo largo del tiempo. Los mapas también se utilizan para mostrar cómo aspectos concretos se distribuyen durante un proceso y proporcionan una forma de filtro en el proceso para este tipo de información.
