Concepto: Categoría personalizada
Las categorías personalizadas se utilizan para categorizar el contenido de acuerdo con los criterios del usuario. Uno de sus usos más importantes es la construcción de vistas.
Descripción principal

Una categoría personalizada es una categoría introducida por un autor del método para estructurar cualquier cantidad de elementos del contenido del método de cualquier tipo de acuerdo con los criterios definidos por el usuario. Las categorías personalizadas se utilizan para categorizar el contenido de acuerdo con los criterios del usuario así como para definir estructuras completas de árbol con categorías anidadas que permiten al usuario la navegación y la exploración sistemáticas del Contenido del método, así como los procesos basados en estas categorías. Esta aplicación de las categorías personalizadas también se denomina Vista.

Ejemplos

Se puede crear una categoría personalizada para organizar de forma lógica el contenido que es relevante para los departamentos de la empresa de desarrollo del usuario: una categoría de "pruebas" puede agrupar todos los roles, productos de trabajo, tareas, patrones de posibilidad y guías que son relevantes para la prueba.

Otro ejemplo pueden ser las categorías que expresan los niveles de licencia del contenido: estas categorías pueden separar el contenido del método de distribución libre del contenido del método que representa una propiedad intelectual y precisa de la adquisición de una licencia para su uso.
