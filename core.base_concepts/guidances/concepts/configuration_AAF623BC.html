Concepto: Configuración del método
Una configuración del método especifica la selección de un subconjunto lógico de una biblioteca de métodos.
Descripción principal

Una configuración del método es una recopilación de plug-ins de método, paquetes del método y procesos seleccionados (consulte Concepto: Patrón de posibilidad, Concepto: Proceso de entrega). Una configuración puede exportarse en su propia biblioteca autónoma cuando incluye el cierre transitivo completo de todos los elementos de los que depende. Una configuración del método define un subconjunto lógico de una biblioteca de métodos.

Aplicaciones

Una configuración acostumbra a crearse en torno a uno o más procesos de entrega. Un proceso de entrega puede ser válido para distintas configuraciones del método, pero cada configuración puede incluir o excluir contenido específico para situaciones concretas. Por ejemplo, puede definirse un proceso de entrega para que incluya contenido de esquemas de desarrollo para distintos tipos de sistemas de gestión de base de datos, como el contenido de RDBMS y OODBMS. Cuando utilicen un proceso de entrega de este tipo, los usuarios pueden seleccionar simplemente el tipo de DBMS empleado en su proyecto y, por lo tanto, excluir todo el contenido relativo a otros tipos de DBMS. Pueden conseguirlo seleccionando una configuración que excluya el contenido respectivo o creando una nueva si no existe ya una de ese tipo.
