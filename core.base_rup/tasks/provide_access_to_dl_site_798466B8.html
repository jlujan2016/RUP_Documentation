Tarea: Proporcionar acceso a un sitio de descargas
En esta tarea se describe cómo poner el producto a disposición de los clientes a través de Internet.
Disciplinas: Despliegue
Objetivo

Garantizar que el producto esté disponible para adquirirlo y descargarlo a través de Internet.

Pasos
Añadir archivos de producto al servidor

En la verdadera tradición web, cualquier número de clientes conectados a Internet (en cualquier momento dado) debe poder acceder a un sitio del host y solicitar archivos, o comprar un producto, a través del servidor. Ello implica que no sólo existe una presencia web de la empresa, sino que la presencia es fiable 24x7 y está habilitada para transacciones de comercio electrónico seguras.

El gestor de despliegue debe poner la unidad de despliegue (el software ejecutable, los archivos de instalación y la documentación) y cualquier otra información pertinente acerca del producto en el servidor en un directorio de acceso predeterminado.

Habilitar el acceso del cliente al producto

El gestor de despliegue debe trabajar con cualquier persona de la organización que sea responsable de publicar y mantener el sitio web de la empresa. El sitio web debe incluir una página que describa el producto con todo el material gráfico de marca correspondiente.  La página del producto debe estar habilitada para proporcionar acceso al producto (unidad de despliegue) y a los archivos relacionados.

El sitio web debe permitir al cliente realizar compras en línea, descargar el producto, así como proporcionar punteros para el soporte del producto y respuestas a las "preguntas más frecuentes".

Habilitar los comentarios de los clientes y las funciones de soporte

De la misma manera que se obtienen comentarios del programa de prueba de versión beta, el sitio web del producto debe configurarse para solicitar comentarios del cliente o para animarle a utilizar programas de actualización.
