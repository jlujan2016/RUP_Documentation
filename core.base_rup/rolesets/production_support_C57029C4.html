Conjunto de roles: Producción y soporte
Este conjunto de roles consta de los roles que no están directamente relacionados con la definición, gestión, desarrollo y prueba de software, pero que se necesitan para dar soporte al proceso de desarrollo de software, o para producir materiales adicionales que necesita el producto final.
