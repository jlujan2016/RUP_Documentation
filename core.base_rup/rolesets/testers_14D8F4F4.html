Conjunto de roles: Verificadores
Este conjunto de roles se encarga fundamentalmente de verificar el software.
Relaciones
Roles
  • Analista de pruebas
  • Diseñador de pruebas
  • Gestor de pruebas
  • Verificador
Descripción principal
 Este conjunto de roles se ocupa de las habilidades específicas que son exclusivas de las actividades de prueba. Observe que hay roles adicionales relacionados con la disciplina de prueba que se construyen sobre las habilidades básicas de otros conjuntos de roles y las amplían. Estos roles adicionales pueden encontrarse en los otros conjuntos de roles organizados por el conjunto de habilidades básicas que amplían (por ejemplo, Gestor, Diseñador, Analista).
