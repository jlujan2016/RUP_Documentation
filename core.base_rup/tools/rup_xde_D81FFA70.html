Descripción principal

IBM® Rational® XDEtm es una herramienta de desarrollo y modelado de componentes gráficos que utiliza el Lenguaje unificado de modelado (UML) estándar del sector.

Las siguientes guías de herramientas comprenden tanto Java Platform Edition de Rational XDE Developer como Microsoft .NET Edition de Rational XDE Developer. Las diferencias entre las dos plataformas se señalarán a continuación.

Visión general

Rational XDE Developer, Java Platform Edition, combina el diseño y el desarrollo de Java en un entorno estrechamente integrado. Este único entorno evita la necesidad de tener que pasar de una herramienta a otra fuera de ese entorno.

Rational XDE mejora la forma de trabajar en muchas maneras. Por ejemplo, le permite realizar lo siguiente:

  • Pasar del análisis y diseño hasta el código dentro del mismo entorno.
  • Especificar sincronización manual o automática entre el modelo y el código.
  • Definir plantillas de código y plantillas de modelo para ahorrar tiempo y ayudar a reforzar los estándares.
  • Crear modelos de UML y de formato libre y validar modelos en conformidad con UML.
  • Crear múltiples modelos dentro del mismo proyecto que puedan rastrearse entre sí.
  • Reutilizar padrones definidos para compartir código en el equipo.

Rational XDE Developer, Java Platform Edition, puede utilizarse con cualquiera de los siguientes productos de la plataforma Websphere Studio:

  • IBM Websphere Studio Workbench (WSW)
  • IBM Websphere Studio Application Developer (WSSAD), similar a WSW pero con características adicionales para Java ™ 2 Enterprise Edition (J2EE) y desarrollo de servicios web.

Ambos son entornos de desarrollo e integración de herramientas compuestos de perspectivas, vistas, editores, menús y otros bloques de construcción del entorno Rational XDE. Por norma, se utilizan diversas perspectivas:

  • Las perspectivas Java contienen editores de Java, visores de clases y muchos otros editores.
  • La perspectiva de modelado contiene el explorador de modelos, el cuadro de herramientas y otras vistas.

Rational XDE realiza prácticamente lo mismo en ambos entornos de producto. En las guías de herramientas encontrará algunas diferencias menores.
