Descripción principal

La aplicación IBM® Rational® Software Architect unifica todos los aspectos del diseño y desarrollo de software en una herramienta sólida y fácil de utilizar. Da soporte a la comprensión, diseño, gestión y evolución de las soluciones y servicios empresariales. El producto incluye todas las características de J2EE, web y servicios web encontrados en IBM® Rational® Application Developer para software de WebSphere. Rational Software Architect se crea sobre la plataforma abierta y ampliable de Eclipse, que optimiza numerosos estándares abiertos del sector. Esto permite a los usuarios crear aplicaciones optimizadas para middleware de IBM, así como las desarrolladas con tecnología middleware de otros proveedores.
