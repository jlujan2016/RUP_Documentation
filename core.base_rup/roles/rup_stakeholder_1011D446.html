Rol: Interesado
Este rol representa a un grupo de interés cuyas necesidades se deben satisfacer con el proyecto. Lo puede desempeñar cualquiera que esté materialmente afectado por el resultado del proyecto.
Conjuntos de roles: Analistas, Roles generales
Relaciones
Descripción principal

Este rol se puede asignar de los modos siguientes:

  • Asigne uno o más miembros del personal para que desempeñen sólo el rol. Esto funciona en grandes equipos, donde existe un número suficiente de expertos en la materia disponibles que tienen conocimiento del dominio significativo que es crítico para el éxito del proyecto.
  • Asigne uno o más miembros del personal para que desempeñen este rol y el Rol: Analista de sistemas o Rol: Especificador de requisitos. Esta estrategia es una buena opción para equipos de prueba pequeños o medianos, y se suele utilizar donde los expertos del dominio pueden desempeñar ambos roles. Debe tener cuidado de dedicar el esfuerzo apropiado para satisfacer ambos roles, y que el experto de dominio tenga las habilidades adecuadas para la adquisición de requisitos y la comunicación.
  • Asigne uno o más miembros del personal para que desempeñen este rol y los roles Rol: Analista de pruebas Rol: Verificador . Esta estrategia es otra opción para equipos de prueba pequeños o medianos, y se suele utilizar donde los expertos del dominio pueden desempeñar ambos roles. Una vez más, debe tener cuidado de dedicar el esfuerzo apropiado para satisfacer ambos roles, y que se facilite formación suficiente a las prácticas de prueba para que este enfoque sea viable.

Si se adopta cualquiera de los últimos enfoques, se recomienda que se asegure de que como mínimo una persona con habilidades especializadas y experiencia se asigne principalmente a los roles indicados aquí como secundarios.

Personal
Habilidades

Este rol requiere experiencia en el tema del dominio o en el área de interés.

En las culturas de algunos proyectos, una persona que desempeña el rol Interesado debe actuar como representante de otra serie de personas privadas de derechos: personas que sólo estarán afectadas materialmente por el resultado del proyecto, pero que no pueden representar directamente sus necesidades por algún motivo. Como tal, el Interesado asignado representante debe poder adquirir información suficiente de los otros miembros del grupo de interés para representar totalmente sus necesidades.
