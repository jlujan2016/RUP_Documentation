Descripción
Este proceso de entrega se diseña para proyectos de gran tamaño. Incluye los flujos de trabajo de iteración para cada una de las cuatro fases de RUP que se han entregado a lo largo de los años, con la adición de la arquitectura orientada a servicios. Incluye el modelo de dominio, del modelado empresarial, así como las actividades de evolución heredada. Este proceso de entrega puede utilizarse tal cual, pero es más probable que desee personalizarlo para que cumpla las necesidades específicas del proyecto.
