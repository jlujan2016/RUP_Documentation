Artefacto: Plan de medidas
Este artefacto define los objetivos de las medidas, las métricas asociadas, y las métricas primitivas que se recopilarán en el proyecto para supervisar su progreso.
Clases de producto de trabajo: Plan
Objetivo

El Plan de medidas de software especifica qué métricas primitivas deben recopilarse y qué métricas deben computarse durante el proyecto para supervisar el progreso, en relación con un conjunto de objetivos de proyecto especificados (consulte el apartado Concepto: Métricas). Se utiliza para recopilar información sobre el proyecto como la entrada de la valoración de estado periódica (consulte Producto de trabajo: Valoración de estado).
