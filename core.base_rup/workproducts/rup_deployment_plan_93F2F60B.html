Artefacto: Plan de despliegue
Este artefacto describe el conjunto de tareas necesarias para instalar y probar el producto desarrollado de forma que se pueda transicionar de forma efectiva a la comunidad de usuario.
Dominios: Gestión de proyectos
Clases de producto de trabajo: Plan
Objetivo

El objetivo del plan de despliegue es garantizar que el sistema llega satisfactoriamente a los usuarios.

El plan de despliegue proporciona una planificación detallada de sucesos, personas responsables y dependencias de sucesos necesarios para garantizar el traslado al nuevo sistema.

El despliegue puede imponer una gran cantidad de cambios y estrés a los empleados del cliente. Por lo tanto, garantizar una transición sueva es un factor clave en la satisfacción del cliente. El plan de despliegue debe minimizar el impacto del traslado del personal del cliente, los sistemas de producción y en la rutina global de la empresa.

Factores clave

Para poderse desplegar, todos los sistemas deben disponer de un plan de despliegue. Si el sistema sólo se construye como prototipo o como prueba de concepto, un plan de despliegue puede no ser necesario.
