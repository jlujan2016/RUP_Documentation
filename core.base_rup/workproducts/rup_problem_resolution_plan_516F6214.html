Artefacto: Plan de resolución de problemas
Este artefacto describe el proceso que se utiliza para reportar, analizar y resolver los problemas que ocurren a lo largo del proyecto.
Clases de producto de trabajo: Plan
Objetivo

Para describir un proceso ordenado para crear informes, analizar y resolver problemas que  ocurren en el proyecto.

Relaciones
Artefacto del contenedor
  • Plan de desarrollo de software
RolesResponsable:
  • Gestor de proyectos
Modificado por:
  • Gestor de proyectos
TareasEntrada a:
  • Compilar el plan de desarrollo de software

  • Desarrollar el plan de garantía de calidad
  • Manejar las excepciones y los problemas
  • Revisar solicitudes de cambio
Salida de:
  • Desarrollar el plan de resolución de problemas
Uso del proceso
  • Gestión de cambios y configuración  >  Gestionar solicitudes de cambio  >  Plan de resolución de problemas
  • Gestión de proyectos  >  Supervisar y controlar el proyecto  >  Plan de resolución de problemas
  • Gestión de proyectos  >  Planificar el proyecto  >  Plan de resolución de problemas
  • Gestión de proyectos  >  Perfeccionar el plan de desarrollo  >  Plan de resolución de problemas
Ilustraciones
Plantillas
  • Problem Resolution Plan
Personalización
Opciones de representación

El plan de resolución de problemas puede formar parte, físicamente, del plan de desarrollo de software, cuando el entorno del proyecto es sencillo. Si las interacciones del proyecto son complejas (revisiones, auditorías, valoraciones, etc.) con muchos interesados, puede tener sentido separar esto.
