Artefacto: Plantillas específicas del proyecto
Estos artefactos son las plantillas para los productos de trabajo que se utilizan en el proyecto.
Clases de producto de trabajo: Proceso
Objetivo

Las plantillas específicas del proyecto se producen para proporcionar soporte a la producción de productos de trabajo.

Personalización
Opciones de representación

Las plantillas que deben seleccionarse varían de un proyecto a otro. Cada plantilla debe personalizarse para que se adecue a las necesidades del proyecto. La personalización puede ir desde insertar el aspecto y la metainformación del proyecto hasta una personalización más amplia del contenido de las plantillas.
