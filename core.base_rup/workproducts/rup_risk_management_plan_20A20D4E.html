Artefacto: Plan de gestión de riesgos
Este artefacto detalla cómo gestionar los riesgos asociados al proyecto. Detalla las tareas de gestión de riesgos que se desarrollarán, las responsabilidades asignadas y cualquier recurso adicional necesario para la actividad de gestión de riesgos. En un proyecto de menor escala, este plan se puede incorporar al plan de desarrollo de software.
Clases de producto de trabajo: Plan
Objetivo

Para garantizar que los riesgos de un proyecto se identifican, analizan, documentan, mitigan, supervisan y controlan adecuadamente.

Descripción
Descripción principal

El plan de gestión de riesgos describe el enfoque que se utilizará para identificar, analizar, dar prioridad, supervisar y mitigar riesgos. Detalla las tareas de gestión de riesgos que se desarrollarán, las responsabilidades asignadas, y cualquier recurso adicional necesario para la actividad de gestión de riesgos. En un proyecto de menor escala, este plan se puede incorporar al Plan de desarrollo de software.

El plan de gestión de riesgos debe actualizarse cuando cambien los riesgos o las estrategias de mitigación.

Personalización
Opciones de representación

La lista de riesgos puede ser suficiente por sí sola en proyectos pequeños; en el caso de proyectos grandes o de alto riesgo en los que se dedicarán recursos importantes para evitar riesgos, es posible que se necesite un plan de riesgos independiente.
