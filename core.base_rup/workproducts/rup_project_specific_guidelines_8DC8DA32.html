Artefacto: Directrices específicas del proyecto
Este producto de trabajo proporciona una orientación prescriptiva sobre cómo realizar una determinada tarea o un conjunto de tareas en el contexto del proyecto.
Clases de producto de trabajo: Proceso
Objetivo

Los miembros del proyecto utilizan las directrices específicas del proyecto cuando efectúan las tareas que se les han asignado. Las decisiones sobre personalización para los productos de trabajo del proyecto deben capturarse en las directrices específicas del proyecto.

Descripción
Esquematización breve

El esquema depende de los productos de trabajo que producirá el proyecto.  Normalmente hay como mínimo una sección para cada producto de trabajo.

Ilustraciones
Ejemplos

  • C++ Programming Guidelines
  • Java Programming Guidelines
  • Use Case Modeling Guidelines
  • CSPS Creative Design Brief - Inception Phase
  • CSPS Design Comps - Inception Phase
Factores clave

Las directrices suelen seleccionarse en un repositorio subyacente controlado por el grupo de proceso de la empresa. Este producto de trabajo es un marcador de posición de las directrices específicas seleccionadas para el proyecto.

Personalización
Opciones de representación

Las directrices que deben proporcionarse varían de un proyecto a otro dependiendo de cuáles estén disponibles en el momento de ejecutar el proyecto. Las directrices pueden ser específicas a la producción de un determinado producto de trabajo o puede describir una técnica en concreto.

Cada directriz debe personalizarse para que se adapte al contexto específico del proyecto.
