Artefacto: Artefactos de instalación
Este artefacto consta del software y de las instrucciones documentas necesarias para instalar el producto.
Clases de producto de trabajo: Solución
Objetivo

El objetivo de los artefactos de instalación es permitir que se instale el producto.

Personalización
Opciones de representación

Los artefactos de instalación pueden incluir los siguientes elementos:

  • Scripts de instalación, archivos de configuración, etc.
  • Instrucciones de instalación

Los artefactos de instalación son necesarios si se van a utilizar programas de instalación para configurar el sistema en el entorno de despliegue. Si el software sólo se despliega una vez (como es el caso en muchos sistemas creados por una empresa para uso interno en un servidor corporativo), los artefactos de instalación pueden omitirse.

En un sistema donde se espera que el usuario instale el producto, las instrucciones de instalación pueden incluirse en la guía del usuario. Para una instalación más complicada donde se necesita personal de servicio cualificado, debe haber una descripción de instalación independiente.
