Artefacto: Elemento de implementación
Estos artefactos son los componentes físicos que forman una implementación, que incluyen archivos y directorios. Incluyen los archivos de código de software (origen, binario o ejecutable), los archivos de datos y los archivos de documentación, como los archivos de ayuda en línea.
Clases de producto de trabajo: Solución
Objetivo

Los elementos de implementación son componentes de una implementación, específicamente las unidades de nivel más bajo de la composición física, sustitución, control de versión y gestión de la configuración. Incluyen los archivos de código de software (origen, binario o ejecutable), los archivos de datos y los archivos de documentación, como los archivos de ayuda en línea.
