Descripción
Descripción principal

El diseño de prueba incorpora todas las decisiones referentes a los elementos estructurales de diseño y sus colaboraciones, necesarios para habilitar las pruebas de los destinos que ya se han identificado. Este producto de trabajo dirige las actividades de implementación de prueba y aumenta el foco de prueba durante el diseño y la implementación.

Esquematización breve

Para cada elemento del destino de la prueba, todos los mecanismos de prueba deben identificarse y describirse y todas las responsabilidades tienen que asignarse a los elementos estructurales de prueba apropiados. Para cada caso de prueba significativo, el diseño debe especificar cómo se realiza en términos de elementos de colaboración, utilizando diagramas de colaboración y/o secuencia.
