Artefacto: Material gráfico del producto
Este artefacto consta de texto (especificaciones impresas) y material gráfico que se utilizarán para comercializar el producto. El material gráfico del producto puede aparecer en el embalaje físico o en un sitio web.
Clases de producto de trabajo: Solución
Objetivo

El material gráfico del producto cumple funciones de marketing. El objetivo del material gráfico del producto es comercializar de forma diferenciada el producto para que se pueda distinguir y resulte atractivo para el cliente potencial. En el caso de productos disponibles a través de Internet, el material gráfico del producto puede ampliar la percepción del sitio web, y la imagen global que el proveedor intenta proyectar.
