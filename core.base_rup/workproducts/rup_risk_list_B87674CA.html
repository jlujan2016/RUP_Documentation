Artefacto: Lista de riesgos
Este artefacto es una lista ordenada de riesgos conocidos y abiertos del proyecto, ordenados en orden decreciente de importancia y asociados con la mitigación específica o las acciones de contingencia.
Dominios: Gestión de proyectos
Clases de producto de trabajo: Datos del proyecto
Objetivo

Para capturar los riesgos percibidos para el éxito del proyecto.

Descripción
Descripción principal

La Lista de riesgos identifica, en orden decreciente de prioridad, los sucesos que pueden llevar a un resultado negativo significativo. Sirve como punto focal para las actividades de proyecto y es la base alrededor de la cual se organizan las iteraciones.

Personalización
Opciones de representación

La lista de riesgos debe capturar los riesgos críticos y graves. Si ve que esta lista supera los 20 puntos, considere cuidadosamente si realmente se tratan de riesgos importantes. El rastreo de más de 20 riesgos es una tarea laboriosa.
