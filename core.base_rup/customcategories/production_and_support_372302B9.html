Producción y soporte
Esta vista reúne todo el contenido del método específico del conjunto de roles de producción y soporte. Este conjunto de roles consta de los roles que no están directamente relacionados con los otros conjuntos de roles que se necesitan para dar soporte al proceso de desarrollo de software, o para producir materiales adicionales que necesita el producto final.
