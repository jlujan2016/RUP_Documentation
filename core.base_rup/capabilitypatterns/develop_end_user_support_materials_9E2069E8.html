Descripción principal

Actualmente, los materiales de soporte para el usuario final, para la mayoría de productos diseñados profesionalmente, se producen como aplicaciones independientes, por ejemplo, sistemas de ayuda o sitios web. La producción de este tipo de aplicaciones suele ser un proyecto en sí mismo y tiene que seguir los siguientes pasos en cada iteración del ciclo vital [HAC97]:

  • Producir un plan de alto nivel para la información que se va a presentar.
  • Especificar el contenido en detalle.
  • Implementar el contenido de la información.
  • Probar y producir los materiales.
  • Evaluar.
