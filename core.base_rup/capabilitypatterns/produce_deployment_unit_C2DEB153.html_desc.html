Descripción

Esta actividad produce la unidad de despliegue. La unidad de despliegue consta de todo el software y los necesarios productos de trabajo adjuntos que se precisan para instalar y utilizar el producto. El producto final consta de la unidad de despliegue, los artefactos de instalación y la documentación del producto.

Las unidades de despliegue se crean para las pruebas de versión beta, así como para el producto final.

Esta actividad depende del conjunto de habilidades de los roles descritos para crear el producto,  los scripts de instalación y el material asociado para el soporte del usuario, de forma que pueda efectivamente entregarse a los usuarios finales.

Utilización
Instrucciones de utilización

Este trabajo se realiza principalmente en las fases de construcción y transición, aunque puede iniciarse en la fase de elaboración, especialmente cuando los aspectos del desarrollo del producto puedan implicar un riesgo de arquitectura significativo. Este trabajo normalmente empieza a la mitad de la iteración y se puede repetir varias veces durante el resto de la iteración.

Este trabajo no se considera opcional, aunque puede ser distinto en cuanto al formato, el estilo y el nivel de ceremonia adecuado al contexto del proyecto.
