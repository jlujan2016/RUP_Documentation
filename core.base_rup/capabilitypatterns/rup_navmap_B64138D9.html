Producto de trabajo: Mapa de navegación
Este artefacto describe la estructura de los elementos de interfaz de usuario del sistema, junto con las vías de navegación potenciales.
Objetivo

Existe un mapa de navegación por sistema. El objetivo del mapa de navegación es expresar las vías de acceso de la interfaz del usuario principal a través del sistema. Estas son las principales vías de las pantallas del sistema y no son necesariamente todas las vías posibles. Se puede considerar un mapa de carreteras de la interfaz de usuario del sistema.

El mapa de navegación sirve de telón de fondo y de enlace entre los guiones gráficos individuales.  Los guiones gráficos describen cómo navega el usuario a través de los elementos de la interfaz de usuario para llevar a cabo las funciones del sistema y el mapa de navegación define cuáles son las vías de navegación válidas. El mapa de navegación transmite la estructura de la interfaz de usuario del sistema, y los guiones gráficos transmiten la dinámica.

El mapa de navegación facilita la visualización de las "pulsaciones" que serán necesarias para que un usuario alcance una pantalla específica.

Personalización
Opciones de representación

Se puede utilizar una gran variedad de representaciones para el mapa de navegación. Algunos ejemplos:

  • Un diagrama en "árbol" jerárquico, donde cada nivel del diagrama muestra el número de pulsaciones que hay que efectuar para llegar a un elemento específico de la interfaz de usuario
  • Gráficos con formato libre con iconos personalizados.

La representación seleccionada y cualquier decisión sobre personalización deben estar documentadas en las directrices específicas del proyecto.
