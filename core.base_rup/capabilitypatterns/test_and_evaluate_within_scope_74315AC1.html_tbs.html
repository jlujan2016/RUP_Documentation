Patrón de posibilidad: Prueba y evaluación [dentro del ámbito]
Este patrón de posibilidad cubre las actividades necesarias para realizar pruebas dentro de un ámbito determinado. El ámbito puede ser un nivel o tipo de prueba determinado, como una prueba funcional. También puede estar limitado en el ámbito de los componentes o partes del mismo, que se han implementado o que está planeado implementarlos durante la iteración.
