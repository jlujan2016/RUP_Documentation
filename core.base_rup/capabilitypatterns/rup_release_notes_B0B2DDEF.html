Producto de trabajo: Notas del release
Este artefacto identifica el contenido, incluyendo los cambios y errores conocidos, de una compilación versionada o de una unidad de despliegue que está disponible para su uso.
Objetivo

El objetivo de las notas del release es describir las características de cada release del producto.

Opciones de representación

Las notas del release generalmente se consideran útiles para producir releases, incluso para los que son estrictamente internos, aunque su formato pueda ser simplificado, casual o informal. Concretamente, el personal de pruebas y de escritura técnica encontrará que las notas del release les resultan útiles para llevar a cabo sus actividades.

Este producto de trabajo normalmente se incluye en el material de soporte para el usuario final.
