Producto de trabajo: Lista de materiales
Este artefacto lista los componentes que constituyen una versión determinada de un producto y dónde se pueden encontrar los componentes físicos. Describe los cambios que se efectúan en la versión y hace referencia a cómo se puede instalar el producto.
Objetivo

Las personas siguientes utilizan las listas de materiales:

  • El gestor de despliegue, para garantizar que todos los elementos necesarios están disponibles para la entrega al cliente.
  • El artista gráfico hace referencia a la lista de materiales como lista de comprobación para garantizar que todos los componentes que van forman parte del paquete del producto y de comercialización están disponibles.
  • La auditoría, contratación y las organizaciones de cliente pueden utilizar la lista de materiales para garantizar que todos los elementos que van a formar parte del producto están contabilizados.

Opciones de representación

Cualquier lista de materiales deberá describir los elementos descritos en el esquema (anterior) y en las plantillas de los productos de trabajo a las que se hace referencia. Es posible que una organización desee expandir la tabla de contenidos. Sin embargo, es importante garantizar que todos los elementos incluidos en la tabla de contenidos puedan encontrarse por referencia o por inclusión directa en el documento de la lista de materiales.
