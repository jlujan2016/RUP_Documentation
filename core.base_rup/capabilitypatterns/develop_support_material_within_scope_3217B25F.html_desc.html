Descripción

Este patrón, o conjunto de actividades, suele repetirse para cada iteración.  

[dentro de Ámbito] en el nombre de este patrón indica que las actividades de este patrón están limitadas por un ámbito concreto, como un manual del usuario determinado, por ejemplo. Este patrón se facilita para utilizarlo varias veces, una para cada 'ámbito' adecuado. Por ejemplo, una vez por cada documento de usuario, o parte de un documento, que se haya planeado para una iteración.
