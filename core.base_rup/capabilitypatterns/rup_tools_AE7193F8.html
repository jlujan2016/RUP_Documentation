Producto de trabajo: Herramientas
Estos productos de trabajo son las herramientas necesarias para proporcionar soporte a la tarea de desarrollo de software.
Objetivo

Un proceso de ingeniería de software requiere herramientas para proporcionar soporte a todas las actividades del ciclo de vida del sistema.
