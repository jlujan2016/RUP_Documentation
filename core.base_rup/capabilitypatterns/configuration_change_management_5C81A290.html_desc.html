Patrón de posibilidad: Gestión de cambios y configuración
Este patrón de posibilidad se encarga de las actividades y el flujo de trabajo de la disciplina de gestión de cambios y configuración.
DescripciónEstructura de desglose de trabajoAsignación de equiposUtilización del producto de trabajo
Objetivo

Para explicar el trabajo que supone la disciplina de gestión de cambios y configuración, las actividades, tareas y productos de trabajo se organizan en un patrón de posibilidad para la disciplina.

Relaciones
Contexto
  • Classic RUP (for large projects)
Descripción

Cada una de las actividades de este patrón de posibilidad representa un objetivo de alto nivel que debe alcanzarse para realizar una configuración y una gestión de cambios eficaces. Las actividades Actividad: Planificar la configuración del proyecto y el control de cambios  y Actividad: Crear entornos de gestión de la configuración (CM) del proyecto se realizan al inicio de un proyecto. El resto se realiza de forma continuada a lo largo del ciclo vital del proyecto.

Notas de utilización

Decida cómo llevar a cabo el flujo de trabajo

Las decisiones siguientes deben tomarse considerando el flujo de trabajo de la disciplina de gestión de cambios y configuración:

  • Decida cómo llevar a cabo el flujo de trabajo observando las actividades de este flujo de trabajo. Estudie el diagrama con sus condiciones de vigilancia y las directrices que se muestran a continuación. Decida las actividades que se van a realizar y su orden.
  • Decida las partes de las actividades de gestión de cambios y configuración que deben llevarse a cabo. Por ejemplo, la Actividad: Gestionar solicitudes de cambio sólo se realiza si va a realizar solicitudes de forma sistemática.
  • Decida en qué punto del ciclo vital del proyecto desea introducir cada parte del flujo de trabajo.
