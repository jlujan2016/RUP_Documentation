Rol: Cualquier rol
Este rol caracteriza las tareas que puede efectuar cualquier miembro del equipo.

Habilidades

El rol Cualquier rol lo puede desempeñar cualquier miembro del equipo del proyecto al que se haya asignado como mínimo uno de los otros roles específicos de RUP. Como tal, las habilidades de los prerrequisitos necesarios para desempeñar este rol son la capacidad de desempeñar como mínimo uno de los otros roles de RUP, y la formación apropiada o la experiencia en las herramientas del entorno de proyecto que se utilizarán para efectuar las tareas de Cualquier rol.

Propuestas de asignación

El rol Cualquier rol lo desempeñarán todos los miembros del equipo del proyecto. Como tal, una vez que se ha asignado como mínimo uno de estos roles a un miembro del equipo, también se les asignará este rol por omisión.
