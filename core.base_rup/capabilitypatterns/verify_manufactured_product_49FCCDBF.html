Tarea: Verificar el producto manufacturado
En esta tarea se describe cómo garantizar que el producto manufacturado está completo y se puede utilizar.

Descripción principal

A veces, esta tarea tarea recibe el nombre de 'primera inspección de artículo'. Sirve como tarea de control de calidad para garantizar que el producto al detalle tiene todos los atributos y productos de trabajo necesarios.  
Pasos
Verificar el producto en la lista de materiales

El gestor de despliegue debe estar satisfecho que el producto manufacturado esté completo. En este caso, completo significa que el producto tiene todos los componentes constituyentes que se indican en la lista de materiales. El gestor de despliegue debe repasar la lista de comprobación e identificar los componentes del producto.

Seguir las instrucciones de instalación del producto

Después de asegurarse de que el producto manufacturado tiene todos los elementos necesarios, el gestor de despliegue debe asegurarse de que el cliente puede cargar y ejecutar el software. En realidad, este paso es una comprobación para verificar la claridad de las instrucciones de instalación, así como el funcionamiento de los productos de trabajo de instalación en el producto final.  

Comprobar la utilización del producto

Después de haber instalado el software, el gestor de despliegue debe revisar las diversas guías de aprendizaje y probar el producto. El gestor de despliegue debe examinar las características y la documentación de ayuda del producto para asegurarse de que todo está correcto.

Como precaución adicional, el gestor de despliegue debe repasar el ejercicio de notificar errores o llamar a la Línea de ayuda para comprobar la capacidad de respuesta.

Enviar el producto a los clientes

Suponiendo que el producto ha pasado las pruebas visuales y de utilización, el gestor de despliegue puede dar la autorización para liberar el producto y ponerlo a disposición para enviarlo a los clientes. El envío del producto está fuera del ámbito de un proceso de entrega de software; sin embargo, se supone que una empresa de desarrollo de productos de software ha establecido la infraestructura y los canales de marketing adecuados para la distribución del producto.
