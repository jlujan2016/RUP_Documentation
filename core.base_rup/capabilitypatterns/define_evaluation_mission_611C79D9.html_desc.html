Actividad: Definir la misión de evaluación
Esta actividad identifica el enfoque adecuado del esfuerzo de la prueba para la iteración, y consigue acordar con los interesados los objetivos correspondientes que dirigirán el esfuerzo de prueba.

Descripción

Para cada iteración, este trabajo se centra fundamentalmente en:

  • Identificar los objetivos para el esfuerzo de prueba y sus entregables
  • Identificar una buena estrategia de utilización de recursos
  • Definir el ámbito y los límites adecuados para el esfuerzo de prueba
  • Esbozar el enfoque que se va a utilizar
  • Definir cómo se supervisará y valorará el progreso.

Personal

Aunque la mayoría de roles implicados en la disciplina de prueba desempeñan un papel en la realización de este trabajo, el esfuerzo se centra fundamentalmente en torno a los roles de gestor de pruebas y analista de pruebas. Las habilidades más importantes que se necesitan para este trabajo incluyen negociación, obtención, estrategia y planificación.

Mientras la mayoría de los recursos para este trabajo se ampliarán en la construcción, deberán asignarse a este trabajo recursos significativos del inicio y de la transición. Como indicación relativa del uso de los recursos de prueba para esta actividad, los porcentajes habituales por fases son: principio - 50%, elaboración - 25%, construcción - 10% y transición - 10%.
Utilización
Instrucciones de utilización

Normalmente tiene lugar hacia el principio de cada iteración antes de iniciar otro trabajo relacionado con las pruebas.

Observe que este trabajo se realiza en cada iteración. Es recomendable que no destine mucho tiempo a la planificación detallada de las tareas de prueba con demasiada antelación respecto a la iteración en la que se realizan. Como norma general, no planifique trabajo de pruebas detallado con más de una iteración de antelación.

El valor principal de la realización de este trabajo es pensar en las distintas preocupaciones y asuntos que afectarán a las pruebas durante el curso de la iteración, y tenga en cuenta las acciones correspondientes que debe llevar a cabo. Como norma general, no destine demasiado tiempo a la presentación de la documentación para dichos aspectos del esfuerzo de prueba.
