Descripción

El material de soporte cubre toda la información que precisará el usuario final para instalar, operar, utilizar y mantener el sistema que se entrega. También incluye material de formación para todas las posiciones distintas que se precisarán a la hora de hacer un uso eficaz del nuevo sistema.

Utilización
Instrucciones de utilización

Aunque este trabajo puede iniciarse ya en la fase de elaboración, normalmente se lleva a cabo en las iteraciones de construcción y transición. Dentro de una iteración, puede iniciarse cuando se cree la primera compilación de software.

La mayoría de software requiere que lo acompañe algún tipo de material de soporte. En casos excepcionales, este material puede excluirse de los materiales entregables de acuerdo con los interesados del proyecto.
