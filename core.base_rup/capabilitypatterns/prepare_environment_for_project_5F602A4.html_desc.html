Actividad: Preparar el entorno para el proyecto
Esta actividad prepara el entorno de desarrollo de un proyecto en el que el entorno de desarrollo incluye tanto el proceso como las herramientas.
DescripciónEstructura de desglose de trabajoAsignación de equiposUtilización del producto de trabajo
Objetivo
La finalidad de esta actividad es convertir el proceso de desarrollo subyacente en un proceso específico del proyecto y preparar la infraestructura de desarrollo para el proyecto.
Relaciones
Actividades principales
  • Entorno
Descripción

La preparación del entorno de desarrollo implica:

  • Definir cómo va a utilizar el proyecto el Proceso de desarrollo configurado.
  • Calificar las selecciones de productos de trabajo con los requisitos de tiempo y de formalidades.
  • Preparar activos específicos del proyecto, como las directrices y las plantillas.
  • Producir una lista de herramientas posibles para su utilización durante el desarrollo.

Estas actividades las realiza mejor un equipo pequeño, compuesto por individuos con sólidos conocimientos generales del proceso de desarrollo. Si la empresa tiene un grupo de proceso dedicado que se centra en el entorno de proceso y desarrollo, el Rol: Ingeniero de proceso suele ser un miembro de este grupo.

Utilización
Instrucciones de utilización

La preparación del entorno para el proyecto se realiza durante la fase inicial. El trabajo posterior que se realiza para mejorar y completar el entorno preparado se describe en la Actividad: Preparar el entorno para una iteración.

Factores clave

Es siempre recomendable emplear algún tiempo en la preparación del entorno de desarrollo de acuerdo con las necesidades específicas del proyecto.

La información preliminar acerca del estado de la empresa de desarrollo suele recopilarse antes del inicio del proyecto y se documenta en Producto de trabajo: Valoración de la empresa de desarrollo.
