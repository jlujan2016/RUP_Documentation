Descripción

La actividad se centra en:

  • Garantizar que los artefactos y sus líneas base asociadas estén disponibles.
  • Determinar si los artefactos necesarios están almacenados en una biblioteca controlada y con una línea base.
  • Dar soporte a las actividades de contabilidad del estado de configuración del proyecto.
  • Facilitar los informes con información de solicitud de cambio, especialmente las actividades relacionadas con el trabajo realizado contra defectos y solicitudes de mejora.
  • Garantizar que los datos se recopilan y se transmiten para realizar el seguimiento del progreso y de las tendencias.

Personal

La persona que desempeña el Rol: gestor de configuración de acuerdo con el plan de gestión de la configuración debe proporcionar informes cuantitativos que pueden utilizarse para valorar el estado del proyecto.

Utilización
Instrucciones de utilización

Este trabajo es aplicable en todo el ciclo vital del proyecto, empezando ya en la fase inicial.

Este trabajo no se considera opcional, aunque según la cultura del proyecto este trabajo variará de un cometido más o menos importante, y su complejidad será mayor cuanto más grande o más distribuido se convierta en equipo de proyecto.
