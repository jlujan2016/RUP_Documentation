Descripción

Para explicar el trabajo que supone la disciplina de prueba, las actividades y productos de trabajo se organizan en un patrón de posibilidad para la disciplina.

Cada actividad representa un objetivo de alto nivel que necesita alcanzarse para realizar una prueba eficaz del producto.

Este patrón de posibilidad puede precisar variaciones basadas en las necesidades específicas de cada iteración y proyecto.

Decida cómo llevar a cabo el flujo de trabajo



Las decisiones siguientes deben tomarse considerando el flujo de trabajo de la disciplina de prueba:

  • Decida cómo llevar a cabo el flujo de trabajo observando las actividades de este flujo de trabajo. Estudie el diagrama con sus condiciones de vigilancia y las directrices que se muestran a continuación.
  • Decida las partes de las actividades de prueba que deben llevarse a cabo. Un aspecto clave del flujo de trabajo de prueba es decidir las dimensiones de calidad que interesan para el proyecto en general, y lo que es más importante, para cada iteración (consulte Concepto: Tipos de pruebas). Decida las combinaciones adecuadas de tipos de pruebas en los que debe centrarse para la iteración actual.
