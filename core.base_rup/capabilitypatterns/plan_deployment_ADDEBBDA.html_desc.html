Actividad: Planear el despliegue

Descripción

La planificación del despliegue requiere un alto grado de colaboración y preparación del cliente. Una conclusión satisfactoria para un proyecto de software puede recibir un grave impacto por parte de factores que se encuentran fuera del ámbito del desarrollo de software, como la compilación, que la infraestructura de hardware no esté en su sitio, o que el personal no esté adecuadamente preparado para el cambio al nuevo sistema.

Para garantizar un despliegue y una transición satisfactorios al nuevo sistema y a las formas de hacer negocio, el plan de despliegue debe contemplar no sólo el software que se va a entregar, sino también el desarrollo del material de formación y el material de soporte para el sistema, para asegurarse de que los usuarios finales pueden utilizar satisfactoriamente el producto de software que se entrega.

Un gestor de despliegue debe ser alguien que comprenda las necesidades operativas del usuario final y que sea capaz de reunir todos los elementos que se necesitan para la realización del producto. El gestor de despliegue ejecuta la prueba de versión beta y, cuando se trata de productos para comercializar, trata con los fabricantes para garantizar que el producto alcanza el nivel de calidad adecuado.

El gestor de despliegue "hace que el producto se entregue" y, por lo tanto, debe estar versado en la infraestructura necesaria y en las necesidades del usuario para garantizar que el producto esté disponible para los usuarios.

Utilización
Instrucciones de utilización

Este trabajo se realiza en una fase inicial de cada iteración, iniciándose normalmente hacia el final de la fase de construcción y continuando durante la transición. Puede iniciarse cuando la planificación de la iteración se ha completado razonablemente para la iteración.

Esta tarea es recomendable siempre que un release ejecutable se vaya a desplegar externamente fuera del equipo de desarrollo principal, ya sea para entregar el software al usuario final o bien por motivos de realización de pruebas.
