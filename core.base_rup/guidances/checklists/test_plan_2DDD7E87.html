Lista de comprobación: Plan de prueba
Esta lista de comprobación le ayuda a garantizar que el plan de prueba está completo.

Descripción principal
Elementos de comprobación
El plan de prueba identifica claramente el ámbito del esfuerzo de prueba

Debe indicar lo siguiente:

  • Estadios y tipos de prueba que se implementarán y se ejecutarán
  • Características de destino de la prueba o funciones que se probarán / no se probarán (según corresponda)
  • Presuposiciones, riesgos o contingencias que puedan afectar o impactar al esfuerzo de la prueba
El plan de prueba identifica claramente los artefactos (y la versión) que se utiliza para generar el contenido del plan de prueba
Cada requisito de proyecto (según se indica en los casos de uso o en las especificaciones suplementarias) tienen como mínimo un requisito asociado para la prueba o una sentencia que justifica porque no es un requisito para la prueba
Todos los requisitos de la prueba se ha identificado y se les ha dado prioridad para cada uno de los diferentes tipos de prueba que se implementan y ejecutan
Una estrategia de prueba clara y concisa se documenta para cada tipo de prueba que se implementará y ejecutará
Para cada estrategia de prueba, se ha indicado claramente la información siguiente:
  • El nombre de la prueba y sus objetivos
  • Una descripción de cómo se implementará y ejecutará la prueba
  • Una descripción de métricas, métodos de medidas y criterios que se utilizarán para evaluar la calidad del destino de la prueba y la completitud de la prueba
Se han identificado todos los recursos necesarios para implementar y ejecutar satisfactoriamente las pruebas, incluido hardware, software y personal
El plan de prueba contiene una planificación o una lista de hitos que identifican el proyecto principal y las tareas relacionadas con la prueba (fechas de inicio y finalización, y/o esfuerzo)
El plan de prueba identifica los artefactos creados por las tareas de prueba, cuando los artefactos estarán disponibles, cómo se distribuirán, su contenido y cómo deben utilizarse
