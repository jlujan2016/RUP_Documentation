Elementos de comprobación
Se ha utilizado el conjunto correcto de atributos de requisitos según lo especificado en el plan de gestión de requisitos
Se han configurado los atributos para cada tipo de requisito para tenerlo en cuenta para lo siguiente, si es aplicable, para cada requisito

Son los siguientes:

  • ¿Estado del seguimiento?
  • ¿Beneficios?
  • ¿Fundamentos?
  • ¿Nivel de esfuerzo que se debe implementar?
  • ¿Tipo y cantidad de cada tipo de riesgo implicado en la implementación?
    • ¿Planificación de riesgo?
    • ¿Riesgo de recurso?
    • ¿Riesgo técnico?
  • ¿Estabilidad del requisito?
  • ¿Release de destino?
  • ¿Asignación?
  • ¿Entrada de marketing?
  • ¿Entrada de desarrollo?
  • ¿Historial de revisión?
  • ¿Ubicación?
  • ¿Motivos para el cambio?
  • ¿Requisitos no concluyentes?
Todas las rastreabilidades se han establecido según lo especificado para el proyecto en el plan de gestión de requisitos
