Lista de comprobación: Script de prueba
Esta lista de comprobación le ayuda a garantizar que se ha comprobado la corrección y la completitud de los scripts de prueba.
Relaciones
Elementos relacionados
  • Script de prueba
Descripción principal
Elementos de comprobación
General
  • ¿Se han implementado suficientes pruebas para alcanzar una cobertura de la prueba aceptable?
  • ¿Los nombres de script de prueba o ID son coherentes con el convenio de denominación del producto de trabajo de prueba?
Automatización de la prueba
  • ¿Cada script de prueba se ha implementado de la forma adecuada según la intención de la estrategia de prueba y la infraestructura de automatización de prueba? Esto se aplica a los scripts de prueba registrados, programados o generados. Tenga en cuenta que debe prestarse atención a la reutilización y al mantenimiento de los scripts de prueba.
  •  ¿Cada script de prueba se ha reproducido y depurado para garantizar que se ejecuta según lo previsto?
    • ¿Cada prueba que contiene un punto de control o lógica de ramificación se ha reproducido y depurado a través de las vía de acceso lógicas posibles?
    • ¿Cada script de prueba se ha implementado para tener en cuenta los cambios constantes esperados en el estado de la aplicación, como los campos de fecha del sistema, números de transacción, etc.?
