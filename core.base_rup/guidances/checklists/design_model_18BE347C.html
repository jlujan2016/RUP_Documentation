Lista de comprobación: Modelo de diseño
Esta lista de comprobación le ayuda a garantizar que el modelo de diseño se especifica correctamente.
Relaciones
Elementos relacionados
  • Modelo de diseño
Descripción principal
Elementos de comprobación
General
  • Los objetivos del modelo se expresan con claridad y son visibles.
  • El modelo tiene un nivel de detalle adecuado para los objetivos del modelo.
  • El uso del modelo de las construcciones de modelado es adecuado para el problema en cuestión.
  • El modelo es lo más simple posible aunque logra los objetivos del modelo.
  • El modelo parece capaz de dar cabida a cambios futuros que sea razonable esperar.
  • El diseño es adecuado para la tarea en cuestión (ni demasiado complejo ni demasiado avanzado).
  • El diseño parece comprensible y de un mantenimiento asumible.
  • El diseño parece implementable.
Capas
  • No hay más de siete capas (dos más o menos).
  • Los fundamentos de la definición de capas se presenta con claridad y se aplica de forma coherente.
  • Los límites de capa se respetan en el diseño.
  • Las capas se utilizan para encapsular los límites conceptuales entre los distintos tipos de servicios y proporcionar abstracciones útiles que facilitan la comprensión del diseño.
