Directriz: Paquete de guiones de uso
Un paquete de guiones de uso es una porción del modelo de guión de uso compuesto de una recopilación semindependiente de guiones de uso estrechamente relacionados. Esta directriz explica cuando es beneficioso descomponer un modelo de guión de uso en paquetes de guiones de uso.

Descripción principal

Explicación

Un modelo estructurado en unidades más pequeñas es más fácil de comprender. Es más fácil mostrar las relaciones entre los componentes principales del modelo si puede expresarlos en términos de paquetes. Un paquete es el paquete de nivel superior del modelo, o se estereotipa como un paquete de guiones de uso. También puede dejar que el cliente decida cómo estructurar los componentes principales del modelo.

  • Si existen muchos guiones de uso o actores, puede utilizar paquetes de guión de uso para estructurar más el modelo de guión de uso. Un paquete de guiones de uso contiene una serie de actores, guiones de uso, sus relaciones y otros paquetes; por lo tanto, puede tener múltiples niveles de paquetes de guiones de uso (paquetes dentro de paquetes).
  • El paquete de nivel superior contiene todos los paquetes de guiones de uso de nivel superior, todos los actores de nivel superior, y todos los guiones de uso de nivel superior.

Utilización

Puede partir un modelo de guión de uso en paquetes de guiones de uso por varios motivos:

  • Puede utilizar los paquetes de guiones de uso para reflejar orden, configuración, o unidades de entrega en el sistema finalizado.
  • La asignación de recursos y la competencia de diferentes equipos de desarrollo puede requerir que el proyecto se divida entre diferentes grupos en diferentes sitios. Algunos paquetes de guiones de uso son adecuados para un grupo, y algunos para una persona, lo que convierte los paquetes en un modo naturalmente eficaz de proceder con el desarrollo. Debe asegurarse, sin embargo, de definir distintas responsabilidades para cada paquete para que el desarrollo se pueda efectuar en paralelo.
  • Puede utilizar paquetes de guiones de uso para estructurar el modelo de guión de uso que refleje los tipos de usuario. Muchos requisitos de cambio se originan en los usuarios. Los paquetes de guiones de uso garantizan que los cambios de un tipo de usuario concreto afectarán sólo a los componentes del sistema que corresponden a ese tipo de usuario.
  • En algunas aplicaciones, cierta información debe estar accesible sólo para unas cuantas personas. Los paquetes de guiones de uso le permiten mantener la privacidad en áreas donde es necesario.
