Directriz: Guiones gráficos
El guión gráfico es una técnica para capturar una descripción lógica y conceptual de la funcionalidad del sistema para un caso de ejemplo específico. Esta directriz destaca las situaciones en que los guiones gráficos pueden resultar útiles y proporciona los modos y los medios para crear guiones gráficos.

Descripción principal

Películas, dibujos, funciones animadas, todos empiezan con guiones gráficos que indican quiénes participan, qué les pasa y cómo pasa.

  • Ayudan a recopilar y perfeccionar los requisitos del cliente de forma sencilla para el usuario.
  • Promueven soluciones de diseño más creativas e innovadoras.
  • Promueven las revisiones de equipo y evita las características que nadie desea.
  • Garantiza que las características se implementan de forma accesible e intuitiva.
  • Facilita el proceso de entrevista - evitando la síndrome de la página en blanco.

En otras palabras, los guiones gráficos significan utilizar una herramienta para ilustrar (y a veces animar) para los usuarios (actores) cómo se adecuará el sistema en la organización, e indicar como se comportará el sistema. Un moderador muestra un guión gráfico inicial al grupo y el grupo proporciona comentarios. El guión gráfico, entonces, evoluciona en "tiempo real" durante el taller. Por lo tanto, necesitará una herramienta gráfica de dibujo para cambiar fácilmente el guión gráfico. Para evitar distracciones, es aconsejable utilizar herramientas sencillas, como gráficos sobre caballetes, una pizarra o Microsoft® PowerPoint®.

Existen dos grupos distintos a utilizar para la creación de guiones gráficos: herramientas pasivas y herramientas activas. Pasivo significa que se muestran imágenes inanimadas, mientras que las herramientas activas tienen capacidades más sofisticadas incorporadas.

Ejemplos de herramientas pasivas para la creación de guiones gráficos:

  • Papel y lápiz
  • Notas Post-it®
  • Constructores de GUI
  • Diferentes tipos de gestores de presentaciones

Algunos ejemplos de herramientas activas para la creación de guiones gráficos son:

  • Apple HyperCard
  • Solutions Etcetera SuperCard
  • Macromedia® Director Shockwave Studio y otras herramientas de animación
  • Microsoft® PowerPoint®

Advertencias y comentarios:

  • Los guiones gráficos deben ser fáciles de crear y de cambiar. Si no ha cambiado nada, no ha aprendido nada.
  • No haga un guión gráfico demasiado bueno. No es un prototipo ni una demostración de la cosa real (percepción "realware").
