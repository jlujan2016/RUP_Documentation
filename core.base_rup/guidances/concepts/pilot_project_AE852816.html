Concepto: Proyecto piloto
En esta guía se describe el concepto básico de un proyecto piloto, qué es y cómo se utiliza.

Descripción principal

Un proyecto piloto es un esfuerzo temporal que se asume para probar la viabilidad de una solución exclusiva del sistema propuesta. Temporal significa que el proyecto tiene una fecha final; exclusivo significa que el resultado final del proyecto es diferente del resultado de otras soluciones del sistema propuestas.

Un proyecto piloto es un proyecto en el que se prueban las ideas nuevas. En el contexto de implementación de procesos y herramientas, esto significa que se prueban procesos y herramientas nuevas.  

Implica que puede añadir recursos adicionales, utilizar personas clave y ajustar el presupuesto y los planes en consonancia. También implica que se debe controlar el proyecto con mucho cuidado, puesto que empezará a utilizar el nuevo proceso y las nuevas herramientas en proyectos reales cuando haya evaluado el proyecto piloto y aprendido con él.    

Seleccione como piloto:

  • Un proyecto de desarrollo de software completo, que se considere de bajo riesgo desde una perspectiva económica y técnica. 

  • La primera iteración completa de un proyecto de desarrollo de software real, con la advertencia de que el aspecto principal es aprender y mejorar el proceso, no desarrollar el software.

Una alternativa puede ser seleccionar un proyecto de desarrollo de software previsto, que se considere de bajo de riesgo desde una perspectiva económica y técnica. De esta forma, se limita el ámbito del proyecto y se convierte en el proyecto piloto. 

Para sacar el máximo partido de un proyecto piloto, es importante que el piloto utilice el entorno de desarrollo real y que haya una presión real sobre el proyecto. 

El proyecto piloto necesita personal clave que:

  • Tenga la atención del público

  • Pueda actuar como mentor

  • Se considere experto en algún área

Es difícil proporcionar unas directrices precisas sobre el tiempo y los recursos. No obstante, las cifras siguientes son comunes a muchos proyectos piloto:

  • Menos de 10 personas

  • Menos de cuatro meses

  • Un mentor 50%

A continuación, se sugiere una definición del término proyecto piloto: 

Un proyecto piloto (nombre) es una actividad planificada como una prueba o un ensayo.
