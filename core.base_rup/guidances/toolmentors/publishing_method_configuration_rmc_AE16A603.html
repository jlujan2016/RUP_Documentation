Instrucciones de la herramienta: Publicación de una configuración del método utilizando Rational Method Composer
En esta guía de la herramienta se describe cómo publicar una configuración del método como un sitio web utilizando Rational Method Composer.
Herramienta: Rational Method Composer

Descripción principal

Una configuración del método es un subconjunto de la biblioteca de métodos definido por el usuario contenido en Rational Method Composer (RMC).

Para que se pueda publicar, se debe crear una configuración. Para obtener información sobre cómo crear una configuración del método, consulte la Guía de la herramienta: Creación de una configuración del método utilizando Rational Method Composer.

Para publicar una configuración existente, seleccione "Configuración>Publicar" y siga las indicaciones. Para obtener más información sobre la publicación de una configuración del método utilizando RMC, consulte el tema de la ayuda en línea que trata sobre la publicación de una configuración del método en una web de métodos. Se puede acceder a toda la ayuda en línea de RMC desde el menú Ayuda.
