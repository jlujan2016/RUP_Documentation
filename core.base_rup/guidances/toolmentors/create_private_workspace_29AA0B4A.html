Descripción principal

Visión general

En esta guía de la herramienta se describe cómo crear un espacio de trabajo privado para el trabajo individual. Un "espacio de trabajo privado" está aislado de los espacios de trabajo de los demás miembros del equipo. Proporciona un lugar en el que el desarrollador puede realizar cambios en el sistema sin ver, ni influir en los cambios de los demás desarrolladores.

En un modelo UCM de ClearCase, los espacios de trabajo privados, denominados espacios de trabajo de desarrollo, le permiten trabajar en actividades de UCM, aislado del resto del  proyecto. Por lo general, cada miembro del proyecto crea su propia área de trabajo de desarrollo al unirse a un proyecto de UCM. Las áreas de trabajo privadas también se pueden crear en cualquier momento durante el desarrollo del trabajo en un proyecto de UCM.

Los espacios de trabajo de desarrollo de UCM constan de una corriente de datos y una vista asociada. Las corrientes de datos de UCM determinan las versiones de los elementos que aparecen en todas las vistas que configura dicha corriente de datos. Las corrientes de datos también mantienen una lista de actividades y líneas base de UCM. Las vistas proporcionan áreas de trabajo para tareas como, por ejemplo, editar y compilar archivos de origen o formatear documentos, entre otras.

Esta guía de la herramienta se aplica al ejecutar Microsoft Windows.

Pasos de la herramienta

En los pasos siguientes se describe cómo crear un espacio de trabajo de desarrollo al unirse a un proyecto de UCM. Los pasos generales son:

  1. Iniciar el asistente para Unir proyecto
  2. Especificar la información de corriente de datos y vista
  3. Adjuntar una vista a una corriente de datos

  Consulte el tema referente al asistente para Unir proyecto de la ayuda en línea de ClearCase para obtener una visión general del asistente. En el tema de ayuda de cada pantalla puede obtener información detallada sobre cada paso del asistente.

Para obtener más información sobre las corriente de datos y las vistas de UCM, consulte los temas de la ayuda en línea de ClearCase que tratan sobre:

  •  Acerca de las corrientes de datos 
  •  Acerca de las vistas

1. Iniciar el asistente para Unir proyecto

  1. Desde la barra de tareas de Windows, seleccione Inicio > Programas > Software Rational > Rational ClearCase > Explorador de proyectos.
  2. En el Explorador de proyectos de ClearCase, pulse la pestaña Cuadro de herramientas > UCM > Unir proyecto. Aparece el asistente para Unir proyecto.
  3. Seleccione el proyecto al que desea unirse y pulse Siguiente para ir al paso siguiente del asistente.

2. Especificar la información de corriente de datos y vista

Especifique el nombre de una corriente de datos, el nombre de una vista, una ubicación de almacenamiento para la vista y el tipo de vista (instantánea) para el área de trabajo de desarrollo. El asistente para Unir proyecto de ClearCase proporciona información por omisión para estos valores.

Nota: El asistente para Unir proyecto también le solicita información necesaria para crear la vista para el área de trabajo de integración del proyecto.

  Para obtener más información sobre el almacenamiento de vistas, consulte el tema que trata sobre cómo elegir una ubicación para un directorio de vistas de instantánea de la ayuda en línea de ClearCase.

3. Adjuntar una vista a la corriente de datos

El asistente para Unir vista le solicita que confirme las elecciones que ha realizado, y une la vista de desarrollo y la corriente de datos que ha especificado. Pulse Aceptar para confirmar las selecciones y continuar.

  Para obtener más información sobre la asociación de vistas y corriente de datos, consulte el tema de la ayuda en línea de ClearCase que trata sobre > cómo unir una nueva vista a una corriente de datos existente.
