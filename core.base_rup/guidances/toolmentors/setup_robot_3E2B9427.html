Descripción principal

Visión general

Esta guía de la herramienta se aplica al ejecutar Microsoft Windows 98/2000/NT 4.0.

Para utilizar Rational Robot para establecer el entorno para la prueba, debe:

  1. Establecer el entorno de prueba para registrar, editar o reproducir scripts de prueba
  2. Establecer las opciones de registro de scripts de la GUI
  3. Establecer las opciones de reproducción de scripts de la GUI

1. Establecer el entorno de prueba para registrar, editar o reproducir scripts de prueba

Tanto el estado del entorno de Microsoft Windows como la aplicación que se somete a prueba pueden afectar a la reproducción de scripts. Si existen diferencias entre el entorno registrado y el entorno de reproducción, se pueden producir problemas de reproducción.

Antes de registrar o editar el script de prueba, asegúrese de que la aplicación que se somete a prueba y todo el software (del entorno de prueba) presentan el estado inicial adecuado. Además, antes de ejecutar las pruebas (reproducir los scripts de prueba), asegúrese de que la aplicación que se somete a prueba y todo el software tienen el mismo estado inicial que presentaban al registrar/editar los scripts de prueba. Todas las aplicaciones y ventanas que estaban abiertas, activas o se visualizaban al iniciar el registro del script deben estar abiertas, activas o visualizarse al iniciar la reproducción. Además, asegúrese de que todos los valores de red apropiados, las bases de datos activas y la memoria del sistema se encuentran en el mismo estado que presentaban al registrar el script.

Al reproducir los scripts de prueba, asegúrese de que establece las opciones de reproducción adecuadas, incluido cómo debe manejar Rational Robot las ventanas activas inesperadas y cómo se debe llevar a cabo la recuperación de errores de mandatos de script (consulte la sección siguiente).

2. Establecer las opciones de registro de scripts de la GUI

Las opciones de registro de scripts de la GUI (interfaz gráfica de usuario) facilitan instrucciones a Rational Robot sobre el modo de manejar determinados objetos, acciones de arrastrar el ratón, valores de ventana, contenido de objetos y la ventana de Robot durante el registro de un script.

Para establecer las opciones de registro de scripts de la GUI:

  1. Visualice el recuadro de diálogo Opciones de registro de la GUI realizando una de las acciones siguientes:
  • Antes de iniciar el registro, pulse Herramientas > Opciones de registro de la GUI.
  • Inicie el registro pulsando el botón Registrar script de la GUI de la barra de herramientas. En el recuadro de diálogo Registro, pulse Opciones.
  1. Establezca las opciones en cada pestaña. Para obtener información detallada sobre cada pestaña, consulte el capítulo que trata sobre el establecimiento de opciones de registro de la GUI del manual Using Rational Robot, o bien, consulte la ayuda en línea de Robot.

General - Especifique cómo identifica/reconoce Robot el contenido de menús y listas, y las acciones de arrastrar el ratón no admitidas. Además, en esta pestaña también puede especificar: el prefijo para la denominación automática de scripts, si Robot guarda y restaura los tamaños y las posiciones de las ventanas activas, si se debe registrar el tiempo de planteamiento, o bien, si se debe registrar un retardo después de pulsar la tecla Intro.

Ventana de Robot - Especifique cómo se debe mostrar la ventana de Robot durante el registro de un script de prueba. Esta pestaña también permite que el usuario especifique las teclas de cambio de modalidad que se deben utilizar para ocultar/volver a mostrar la ventana de Robot y conmutar entre las modalidades de registro orientado a objetos y registro de bajo nivel.

Orden de reconocimiento de objetos - Se utiliza para cambiar la prioridad de métodos de reconocimiento de objetos para tipos de objetos individuales antes o durante el registro.

Para obtener información sobre cómo registrar un script, consulte la Guía de la herramienta: Creación de scripts de prueba utilizando Rational Robot.

3. Establecer las opciones de reproducción de scripts de la GUI

Las opciones de reproducción de scripts de la GUI facilitan instrucciones a Rational Robot sobre cómo reproducir scripts de la GUI. Puede establecer dichas opciones antes de iniciar la reproducción o al principio del proceso de reproducción.

Para establecer las opciones de reproducción de la GUI:

  1. Visualice el recuadro de diálogo Opciones de reproducción de la GUI realizando una de las acciones siguientes:
  • Antes de iniciar la reproducción, pulse Herramientas > Opciones de reproducción de la GUI.
  • Inicie la reproducción pulsando el botón Reproducir script de la barra de herramientas. En el recuadro de diálogo Reproducir, pulse Opciones.
  1. Establezca las opciones en cada pestaña.  

    Para obtener información detallada sobre cada pestaña, consulte el capítulo que trata sobre el establecimiento de opciones de reproducción de la GUI del manual Using Rational Robot, o bien, consulte la ayuda en línea de Robot.

  2. Reproducir - Especifique el retardo entre los mandatos y las pulsaciones de tecla, si se debe utilizar el tiempo de planteamiento registrado y los retardos de teclado, si se deben saltar los puntos de verificación, si se debe mostrar el recuadro de resultados de confirmación y lo que le sucede a la ventana de Robot durante la reproducción. Para obtener información, pulse el botón Ayuda del recuadro de diálogo.

    Registro - Especifique los resultados que se deben guardar en el registro, si el registro se debe mostrar automáticamente después de la reproducción, si desea que se le pregunte antes de sobrescribir el registro y si se debe utilizar información de registro por omisión, o bien, la información que proporciona el usuario.

    Coincidencia de títulos - Especifique cómo debe hacer coincidir Robot los títulos de las ventanas capturados durante el registro con títulos encontrados durante la reproducción. Para obtener información, pulse el botón Ayuda del recuadro de diálogo.

    Estado de espera - Especifique los valores de tiempo de espera y reintento por omisión durante la reproducción.

    Ventanas activas inesperadas - Especifique cómo debe responder Robot ante ventanas activas inesperadas.

    Recuperación de errores - Especifique cómo se debe recuperar Robot ante anomalías de punto de verificación y mandatos de script durante la reproducción de los scripts.

    Trampa - Especifique la información que debe capturar el programa de utilidad Trap sobre faltas de protección general que se producen durante la reproducción y el método de recuperación.

  3. Pulse Aceptar.

Para obtener información sobre la reproducción de un script, consulte la Guía de la herramienta: Ejecución de pruebas con Rational Robot.
