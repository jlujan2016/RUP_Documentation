Instrucciones de la herramienta: Creación de una línea base de un proyecto de Rational RequisitePro
En esta guía de la herramienta se describe cómo crear una línea base de requisitos utilizando Rational Administrator y Rational ClearCase.

Descripción principal

Visión general

Es una buena idea crear una línea base de un proyecto de requisitos en cada objetivo de proyecto. Una línea base crea un objeto de ClearCase que representa una configuración estable del proyecto de Rational RequisitePro® en un momento determinado. Este objeto se puede utilizar para crear un nuevo proyecto.

Un proyecto de RequisitePro consta de una base de datos de requisitos, vistas del proyecto y documentos que contienen requisitos. Los requisitos se pueden crear directamente en una vista del proyecto o en un documento de Microsoft® Word en RequisitePro. Todos los requisitos se almacenan en la base de datos, independientemente de su ubicación en el proyecto. Puede crear una línea base de cualquier proyecto de RequisitePro, sin tener en cuenta si este utiliza una base de datos de Oracle, Microsoft Access o SQL Server.

Nota: para obtener información sobre el archivado de un proyecto de RequisitePro, consulte el apartado Guía de la herramienta: Archivado de requisitos utilizando RequisitePro.

Creación de una línea base de proyecto con Gestión unificada de cambios

Gracias al modelo UCM (Gestión unificada de cambios), los administradores de proyectos de RequisitePro pueden crear líneas base de proyectos de RequisitePro por medio de la realización de tareas en ClearCase y Rational Administrator.

Antes de crear una línea base de un proyecto de RequisitePro, debe cumplir las condiciones previas siguientes:

  • El proyecto de RequisitePro se debe haber asociado a un proyecto Rational habilitado para UCM en Rational Administrator.
  • El proyecto de RequisitePro debe estar cerrado, sin usuarios activos.

Para crear una línea base de un proyecto de RequisitePro:

  1. Abra Rational Administrator.
  2. Pulse el botón derecho del ratón sobre el proyecto del administrador y, a continuación, pulse Configurar.
  3. En el recuadro de diálogo Configurar proyecto del área Activos de los requisitos, pulse Incorporar todos.
  4. En los mensajes que aparecen a continuación, confirme su selección y escriba una actividad para describir la línea base.
  5. Una vez que se haya completado la operación Incorporar todos, cierre el recuadro de diálogo Configurar proyecto.
  6. En el menú Herramientas de Rational Administrator, pulse Explorador de proyectos de Rational ClearCase. Se abre el Explorador de ClearCase.
  7. Pulse el botón derecho del ratón sobre la corriente de datos de integración Activos de los requisitos de RequisitePro y, a continuación, pulse Crear línea base en el menú emergente.
  8. Escriba un título de línea base, seleccione Completo para el tipo de línea base, seleccione Ver contexto y, a continuación, pulse Aceptar.
