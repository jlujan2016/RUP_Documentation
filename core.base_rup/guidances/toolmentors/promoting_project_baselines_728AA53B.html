Instrucciones de la herramienta: Promoción de líneas base de proyectos utilizando Rational ClearCase
En esta guía de la herramienta se describe cómo asignar niveles de promoción a líneas base utilizando UCM (Gestión unificada de cambios) de Rational ClearCase.
Herramienta: Rational ClearCase

Descripción principal

Visión general

En ClearCase, generalmente, las líneas base de UCM representan una configuración estable para un componente.

Las líneas base se asignan a niveles de promoción que se puede utilizar para indicar la calidad o el grado de completitud del trabajo que representa una línea base. Por lo general, el nivel de promoción indica un nivel de prueba. UCM proporciona un conjunto ordenado de niveles de promoción por omisión, y también ofrece soporte para niveles de promoción definidos por el usuario. La acción de cambiar el nivel de promoción de una línea base se denomina promoción o degradación de la línea base.

Una línea base recomendada es aquella que se ha aprobado para uso general. Las líneas base recomendadas se utilizan en operaciones de cambio de base. Cuando se cambia la base de una corriente de datos, se reconfigura con las líneas base recomendadas de la corriente de datos de destino. Uno de los usos de los niveles de promoción consiste en asignar un conjunto de líneas base con el mismo nivel de promoción como líneas base recomendadas para una corriente de datos determinada.

Esta guía de la herramienta se aplica al ejecutar Microsoft Windows.

Pasos de la herramienta

En estos procedimientos se describe cómo cambiar el nivel de promoción de una línea base y cómo convertir líneas base con un nivel de promoción determinado en líneas base para la corriente de datos:

  1. Cambiar el nivel de promoción de una línea base
  2. Basado en el nivel de promoción, establecer una línea base como línea base recomendada para la corriente de datos

Para obtener más información, consulte la publicación IBM Rational ClearCase Guide to Managing Software Projects.

1. Cambiar el nivel de promoción de una línea base

  1. En el Explorador de proyectos de ClearCase, seleccione una corriente de datos. Pulse Archivo > Propiedades para que se muestre la hoja de propiedades de la corriente de datos.
  2. Pulse la pestaña Líneas base.
  3. En el recuadro Componentes, seleccione el componente cuya línea base desea promocionar. Las líneas base del componente se muestran en el recuadro Líneas base. Seleccione la línea base y pulse Propiedades para que se muestre la hoja de propiedades de la línea base.
  4. En la pestaña General, pulse la flecha abajo de la lista Nivel de promoción para visualizar todos los niveles de promoción disponibles. Seleccione el nuevo nivel de promoción y, a continuación, pulse Aceptar.

2. Basado en el nivel de promoción, establecer una línea base como línea base recomendada para la corriente de datos

  • Pulse Inicio > Programas > Software Rational > Rational ClearCase > Explorador de proyectos.
  • Seleccione una corriente de datos.
  • Pulse Herramientas > Líneas base recomendadas para que se muestre una lista de las líneas base recomendadas actuales para la corriente de datos.
  • En el área Generar lista, seleccione un nivel de promoción de la lista desplegable.
  • Pulse Generar lista para convertir las líneas base más recientes del o por encima del nivel de promoción en líneas base recomendadas.   
  • Pulse Aceptar.
