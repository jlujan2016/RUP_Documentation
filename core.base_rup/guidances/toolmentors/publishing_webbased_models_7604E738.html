Instrucciones de la herramienta: Publicación de modelos de Rational Rose basados en web utilizando el publicador de web
En esta guía de la herramienta se describe cómo utilizar Rational Rose Web Publisher para crear una versión basada en web (HTML) de un modelo de Rose para que la visualicen los demás utilizando un navegador como, por ejemplo, Microsoft Internet Explorer o Netscape Navigator.

Descripción principal

Visión general

Al publicar versiones basadas en web de los modelos de Rose, se permite que los demás examinen de modo no secuencial, busquen y naveguen en los modelos. Por ejemplo, puede publicar iteraciones sucesivas de un modelo en desarrollo para revisar o compartir información. Rose Web Publisher vuelve a crear elementos de modelo de Rose, incluidos diagramas, clases, paquetes, relaciones, atributos y operaciones. Una vez que se han publicado, los enlaces de hipertexto le permiten recorrer el modelo como lo haría en Rose. Puede controlar lo que incluye Rose Web Publisher estableciendo diversas opciones. Por ejemplo, puede seleccionar las vistas del modelo que desea que se publiquen, el nivel de detalle que incluir, la notación que utilizar y el formato de gráficos para los diagramas de Rose. Observe que, además, si los guiones de uso del modelo tienen documentos de guiones de uso de RequisitePro adjuntos, Rose Web Publisher conserva los documentos y muestra versiones HTML de estos en las páginas web publicadas.

Pasos de la herramienta

Para publicar un modelo existen dos procedimientos:

  1. Utilizar diálogos del publicador de web
  2. Utilizar una interfaz de proceso por lotes de línea de mandatos

1. Utilizar diálogos del publicador de web

El procedimiento más sencillo para generar un modelo basado en web consiste en utilizar el complemento del publicador de web disponible en el menú Herramientas de Rational Rose. Un diálogo del publicador de web proporciona un navegador que le permite seleccionar los elementos de modelo que desea publicar. Además, puede especificar si las clases que publica deben incluir información de superclase y el formato de gráficos que se debe utilizar para los diagramas (por ejemplo, .jpg o .gif). La característica Vista preliminar le permite iniciar el navegador por omisión y ver el modelo publicado directamente de Rose Web Publisher. 

Para publicar un modelo, siga los pasos que se indican a continuación:

  1. En Rational Rose, abra el modelo que desea publicar.
  2. En el menú Herramientas, pulse Publicador de web.
  3. Utilice el navegador para seleccionar los elementos de modelo que desea publicar, y seleccione las opciones de publicación que desea aplicar.
  4. Cuando esté preparado para publicar el modelo, pulse Publicar. 
  5. Pulse Vista preliminar para abrir el navegador web por omisión y ver el modelo publicado. 

Para obtener más información sobre el publicador de web, consulte la ayuda en línea.

2. Utilizar una interfaz de proceso por lotes de línea de mandatos

Rational Rose Web Publisher tiene una interfaz de línea de mandatos que le permite publicar modelos utilizando un procesador por lotes, que resulta especialmente útil si se desea publicar modelos de modo automático en una planificación rutinaria. Puede controlar el nivel de detalle que desea publicar al definir parámetros de publicación en un archivo de inicialización. Al llamar al procesador del publicador de web, debe proporcionar el nombre del archivo de inicialización. El procesador, rosewpbatch.exe, se encuentra en el subdirectorio rosewp del directorio de instalación de Rose. La sintaxis de llamada del procesador por lotes es la siguiente: 

rosewpbatch.exe su_nombreArchivo_inicialización.ini 

Ejemplo: c:\rose\rosewp\rosewpbatch.exe c:\mybatchfiles\testbatch.ini

Siendo su_nombreArchivo_inicialización.ini el nombre del archivo de inicialización que ha creado. Incluya la vía de acceso completa del archivo .ini en la llamada al procesador. Si el procesador encuentra errores, éstos se registran en el archivo rosewpbatch.err que se encuentra en el mismo directorio que el procesador. Para visualizar el contenido, puede utilizar cualquier editor de texto.

Consulte en la ayuda en línea del publicador de web los parámetros y la sintaxis que debe proporcionar en el archivo de inicialización.
