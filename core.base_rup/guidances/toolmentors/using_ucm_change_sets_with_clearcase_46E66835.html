En esta guía de la herramienta se describe cómo utilizar conjuntos de cambios con actividades UCM (Gestión unificada de cambios) para crear y registrar cambios.

Visión general

En el diagrama siguiente se ilustra el flujo de trabajo de UCM. En esta guía de la herramienta se tratan las áreas sombreadas.

Flujo de trabajo de UCM

Nota sobre terminología: Una actividad de UCM de ClearCase correlaciona exactamente con un pedido de trabajo de RUP. No se debe confundir con el concepto de RUP de una actividad.

Una actividad de UCM hace un seguimiento de las versiones que se crean al trabajar en una tarea de desarrollo. Una actividad incluye un titular de texto que describe la tarea, el ID de usuario del creador de la actividad y un conjunto de cambios que identifica todas las versiones que se crean al trabajar en la actividad. Cuando esté preparado para modificar archivos de origen, debe establecer la vista de desarrollo para una actividad.

Antes de modificar los archivos, debe establecer una actividad para la vista de desarrollo. 

Esta guía de la herramienta se aplica al ejecutar Microsoft Windows. 

Pasos de la herramienta

Para trabajar con actividades de UCM, debe:

  1. Crear o buscar y establecer una actividad
  2. Extraer y modificar versiones
  3. Incorporar el trabajo

1. Crear o buscar y establecer una actividad 

Cuando extrae archivos o directorios de un proyecto de UCM, se le solicita que especifique una actividad para hacer el seguimiento del trabajo.

Cree una nueva actividad y establézcala en la vista actual

  1. En un recuadro de diálogo Extraer, Incorporar o Añadir al control de origen, pulse Nuevo.
  2. En el recuadro de diálogo Nueva actividad, entre el titular de una actividad.
  3. Pulse Aceptar.
  4. Para añadir la actividad a la corriente de datos y establecerla en la vista, pulse Aceptar.

Busque y establezca una actividad

Las actividades se mantienen entre sesiones de trabajo. Para buscar una actividad existente y establecerla en la vista de desarrollo, siga los pasos que se indican a continuación:  

En el Explorador de ClearCase:

  1. En el Explorador de ClearCase, pulse la pestaña Vistas. 
  2. Pulse la página del proyecto y, a continuación, seleccione la vista de desarrollo. 
  3. En el panel Carpeta, pulse Mis actividades.
  4. En el panel Detalles, marque el recuadro de diálogo que se encuentra junto a la actividad que desea establecer.

En los recuadros de diálogo de ClearCase:

  1. En el recuadro de diálogo Extraer, Incorporar o Añadir al control de origen, realice una de las acciones siguientes:
  • Elija una actividad de la lista Actividad.
  • Pulse Examinar y elija otros criterios de selección.
  1. Pulse Aceptar. ClearCase establece la vista para la actividad que se ha seleccionado actualmente en la lista Actividad.

2. Extraer y modificar versiones 

Antes de modificar archivos de origen, vaya a la vista Desarrollo y extráigalos. La extracción permite que se pueda escribir en las versiones de archivo o directorio de la vista.  

  1. En el Explorador de ClearCase, seleccione archivos o directorios y, a continuación, pulse el botón derecho del ratón sobre la selección y pulse Extraer. 
  2. ClearCase abre el recuadro de diálogo Extraer con la actividad establecida actualmente, seleccionada en la lista Actividad. Puede aceptar la selección, elegir otra actividad o crear una nueva actividad.
  3. En el recuadro Comentario, describa los cambios que ha planificado. Los comentarios se muestran en el navegador Propiedades de la versión y en el navegador Historial del elemento.
  4. Realice una de las acciones siguientes:
  • Para extraer sólo un archivo, pulse Aceptar.
  • Para utilizar los valores actuales de este recuadro de diálogo para todos los elementos seleccionados, pulse Aplicar a todos.
  • Para interrumpir el proceso de extracción, salir y dejar todos los elementos restantes incorporados, pulse Cancelar.

3. Incorporar el trabajo 

Cuando desee mantener un registro del estado actual de un archivo, incorpórelo. La incorporación de archivos o directorios añade nuevas versiones a la VOB. La actividad establecida actualmente registra la información de la versión.  

Después de una incorporación, la vista se mantiene establecida en la actividad actual. 

  1. En el Explorador de ClearCase, seleccione archivos o directorios extraídos. 
  2. Pulse el botón derecho del ratón sobre la selección y, a continuación, pulse Incorporar.
  3. En el recuadro Comentario, puede sobrescribir o añadir lo que desee al comentario que ha entrado al extraer el archivo o la carpeta.
  4. Realice una de las acciones siguientes:
  • Para incorporar sólo un archivo, pulse Aceptar.
  • Para utilizar los valores actuales de este recuadro de diálogo para todos los elementos seleccionados, pulse Aplicar a todos.
  • Para interrumpir el proceso de incorporación, salir y dejar todos los elementos restantes extraídos, pulse Cancelar.
