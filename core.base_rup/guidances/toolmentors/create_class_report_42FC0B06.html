Instrucciones de la herramienta: Creación de un informe de clase utilizando Rational SoDA
En esta guía de la herramienta se describe cómo utilizar Rational SoDA para crear un informe de clase.
Herramienta: Rational SoDA
Descripción principal

Visión general

SoDA automatiza la generación del informe, de forma que se pueda crear rápidamente y con precisión. Puede generar un informe de clase con la versión de Microsoft® Word® o Adobe® FrameMaker® de SoDA.

Información relacionada con Rational Unified Process:

  • Informe: Clase
  • Producto de trabajo: Clase de diseño  

Esta guía de la herramienta se aplica al ejecutar Windows 2000, NT 4.0, Windows XP, Solaris o HP-UX.

Para crear un informe de clase utilizando Rational SoDA, siga el procedimiento que se indica para su versión del producto:

  • Utilizar Rational SoDA/Word para generar un informe de clase
  • Utilizar Rational SoDA/FrameMaker para generar un informe de clase

Utilizar Rational SoDA/Word para generar un informe de clase

  1. En un diagrama de clase en Rational Rose, seleccione la clase que desea documentar.
  2. Pulse Informe > Informe SoDA.
  3. Cuando aparezca la lista de informes disponibles en SoDA, seleccione Informe de clase de Rational Unified Process. Si este informe no está en la lista, cancele y asegúrese de que ha seleccionado una clase del diagrama de clase.
  4. Pulse Aceptar para generar el informe.

Utilizar Rational SoDA/FrameMaker para generar un informe de clase

  1. En la barra de botones de FrameMaker, pulse Nuevo. Efectúe una doble pulsación en SoDA y, a continuación, efectúe una doble pulsación en RoseDomain y elija RUPClassReport.fm.
  2. Edite el conector y entre el nombre del archivo de modelo, el paquete y la clase.
  3. Pulse Archivo > Guardar como para guardar la plantilla en un directorio personal o en el directorio del proyecto. Es posible que desee cambiar el nombre de la plantilla de modo que refleje el nombre de la clase; por ejemplo, LoanApplicationReport.fm.
  4. Pulse SoDA > Generar documento.
  5. Revise el documento generado.

La próxima vez que desee generar este documento, simplemente, abra el documento y seleccione SoDA=>Generar documento.
