Descripción principal

Visión general

ClearQuest almacena las solicitudes de cambio en registros de base de datos. El administrador de ClearQuest puede crear tipos de registros diferentes para distintos objetivos y proyectos. Cada tipo de registro puede tener requisitos de datos y campos exclusivos.

Gracias a ClearQuest, los usuarios pueden enviar, modificar, hacer un seguimiento y representar de modo gráfico las solicitudes de cambio fácilmente, mientras se mueven a través del sistema de gestión de solicitudes de cambio.

El administrador de ClearQuest puede crear un conjunto de tipos de registro personalizado. Puede utilizar un tipo de registro como, por ejemplo, un defecto, para todas las solicitudes de cambio, o bien, puede utilizar tipos de registro diferentes para objetivos distintos como, por ejemplo, solicitudes de mejora o solicitudes de documentación, entre otros.
