Instrucciones de la herramienta: Definición de notificaciones de cambio y revisión utilizando Rational ClearQuest
En esta guía de la herramienta se explica cómo establecer Rational ClearQuest® para notificar a los usuarios o grupos de usuarios cuando cambia el estado de una solicitud de cambio.

Descripción principal

Visión general

ClearQuest aprovecha el correo electrónico de dos modos: envío/modificación de correo electrónico (usuario a base de datos) y notificación de correo electrónico (base de datos a usuario).

Envío de correo electrónico: El administrador de ClearQuest utiliza el lector de correo electrónico de Rational para permitir que los usuarios envíen o modifiquen registros a través del correo electrónico. Rational Mail Server se debe ejecutar en la misma máquina que la base de datos de ClearQuest con el objeto de garantizar que el envío de correo electrónico siempre esté disponible.

Notificación de correo electrónico: El administrador de ClearQuest puede configurar ClearQuest para enviar correo electrónico que notifique a los usuarios las solicitudes de cambio. El administrador utiliza el tipo de registro Reglas de correo electrónico para determinar el contenido, los destinatarios y las condiciones del correo electrónico. Además, el administrador o los usuarios pueden permitir que sus máquinas cliente reciban notificaciones de correo electrónico. Incluso los clientes web de ClearQuest pueden recibir notificaciones de correo electrónico.

Pasos de la herramienta

1. Establecer el envío de correo electrónico con ClearQuest Designer

Consulte la ayudan en línea de ClearQuest Designer: Ayuda > Contenido e índice > Administrar el correo electrónico de ClearQuest > Habilitar el envío de correo electrónico

2. Establecer la notificación de correo electrónico con ClearQuest Designer

Consulte la ayuda en línea de ClearQuest Designer: Ayuda > Contenido e índice > Administrar el correo electrónico de ClearQuest > Habilitar la notificación de correo electrónico automática.

3. Recibir la notificación de correo electrónico en el cliente de ClearQuest

Consulte la ayuda en línea de ClearQuest: Ayuda > Contenido e índice > Administrar el correo electrónico de ClearQuest> Habilitar las posibilidades de correo electrónico del cliente de ClearQuest.

4. Establecer la notificación de correo electrónico para clientes web de ClearQuest

Consulte el apartado Instalación de Rational ClearQuest.
