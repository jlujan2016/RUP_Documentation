Descripción principal

Un plug-in de método se utiliza para empaquetar todos los cambios (adiciones, supresiones y ampliaciones, por ejemplo) para procesos y contenido del método existentes. 

Para obtener información sobre cómo crear un plug-in de método, consulte la hoja de orientación "Crear un plug-in de método propio".  Se puede acceder a todas las hojas de orientación de Rational Method Composer seleccionando Ayuda>Hoja de orientación.

Una vez que se ha creado el plug-in, se debe rellenar con contenido:

  • Para obtener información sobre el contenido del método, consulte la Guía de la herramienta: Desarrollo del contenido del método utilizando Rational Method Composer. 
  • Para obtener información sobre el desarrollo de procesos, consulte la Guía de la herramienta: Desarrollo de procesos utilizando Rational Method Composer.
