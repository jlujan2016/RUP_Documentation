Instrucciones de la herramienta: Comparación de líneas base utilizando Rational ClearCase
En esta guía de la herramienta se describe cómo comparar dos líneas base de un componente UCM con el navegador del árbol de componentes de Rational ClearCase.

Descripción principal

Visión general

En ClearCase, una línea base UCM representa, generalmente, una configuración estable para un componente.  Una línea base identifica actividades y una versión de cada elemento visible en uno o más componentes.  El navegador del árbol de componentes de ClearCase es una GUI que muestra el historial de líneas bases de un componente. Lo puede utilizar para comparar el contenido de dos líneas base.

Esta guía de la herramienta se aplica al ejecutar Microsoft Windows.

Pasos de la herramienta

Para comparar dos líneas base de un proyecto de UCM debe:

  1. Iniciar el Explorador de ClearCase
  2. Abrir el navegador del árbol de componentes para un componente especificado
  3. Seleccionar una línea base y elegir parámetros de comparación

Para obtener más información, consulte el manual de ClearCase Managing Software Projects.

1. Iniciar el Explorador de ClearCase

Para localizar el componente cuyo historial de líneas base desea ver, lleve a cabo este paso:

  • Desde la barra de tareas de Windows, pulse Inicio > Programas > Software Rational > Rational ClearCase > Explorador de ClearCase.

2. Abrir el navegador del árbol de componentes para un documento especificado

  • Pulse Buscar líneas base en el menú emergente del componente para iniciar el navegador del árbol de componentes de ClearCase.
  • Pulse la carpeta Componentes del proyecto para que se muestre una lista de componentes.
  • Pulse el botón derecho del ratón sobre un componente para visualizar un menú emergente.
  • Pulse Buscar líneas base para iniciar el navegador del árbol de componentes.

Consulte el tema que trata sobre el navegador del árbol de componentes de ClearCase de la ayuda en línea de ClearCase.

3. Seleccionar una línea base y elegir parámetros de comparación

  • Seleccione una de las líneas base visualizadas como base de la comparación.
  • En el menú Herramientas, elija uno de los elementos de menú Comparar.... Se le solicita que proporcione una segunda línea base para utilizarla en la comparación.
  • Puede ver diferencias en términos de actividades o de versiones. Pulse la pestaña adecuada para elegir lo que desea.
