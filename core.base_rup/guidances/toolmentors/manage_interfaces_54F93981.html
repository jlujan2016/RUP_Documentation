Descripción principal

Visión general

Se deben utilizar uno o más paquetes para gestionar las interfaces de un sistema. Para crear los paquetes, consulte la Guía de la herramienta: Gestión del modelo de diseño.

Para utilizar Rational Rose para crear interfaces, debe:

  1. Crear la interfaz
  2. Crear operaciones de interfaz

1. Crear la interfaz

Una interfaz especifica las operaciones visibles externamente de una clase o componente, y no tiene una implementación propia. Por lo general, una interfaz sólo especifica una parte limitada del comportamiento de una clase o componente. Las interfaces pertenecen a la vista lógica, pero pueden ocurrir en diagramas de clases, guiones de uso y componentes.

Para obtener más información, consulte el tema de la ayuda en línea de Rational Rose que trata sobre las interfaces.

2. Crear operaciones de interfaz

Para crear una interfaz en una interfaz, infórmese sobre las clases en la Guía de la herramienta: Gestión de clases, siguiendo los mismos pasos que se han presentado para la clase y aplicándolos a la interfaz.
