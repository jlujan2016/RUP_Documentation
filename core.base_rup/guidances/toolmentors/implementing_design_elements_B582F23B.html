Instrucciones de la herramienta: Implementación de elementos de diseño utilizando Rational XDE Developer
En esta guía de la herramienta se describe cómo realizar la implementación de elementos de diseño con Rational XDE Developer.
Herramienta: Rational XDE Developer
Relaciones
Elementos relacionados
  • Implementar elementos de diseño
Descripción principal

En esta guía de la herramienta se da por supuesto que se ha creado un conjunto de modelos estructurados de acuerdo con las directrices de la estructura de modelo de XDE y, en especial, modelos para la implementación.

Una implementación se puede compilar editando código fuente e invirtiendo la ingeniería de un modelo de código, o bien, modelando el código en Rational XDE (por lo general, desarrollando el modelo de diseño creado previamente) y generando código fuente.

Para crear elementos de modelo, consulte las guías de la herramienta para las actividades de diseño. Tenga en cuenta que, con frecuencia, los elementos de modelo de implementación se crean moviendo un elemento del modelo de diseño al modelo de implementación. El elemento puede seguir teniendo referencias del modelo de diseño. Consulte el apartado Concepto: Correlación de diseño a código.
