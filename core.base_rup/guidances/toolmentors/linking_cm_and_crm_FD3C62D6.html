Instrucciones de la herramienta: Enlace de la gestión de la configuración y la gestión de solicitudes de cambio utilizando Rational ClearQuest y Rational ClearCase
En esta guía de la herramienta se describe cómo permitir la utilización de Rational ClearQuest con proyectos UCM (Gestión unificada de cambios) de Rational ClearCase.
Herramienta: Rational ClearCase
Relaciones
Elementos relacionados
  • Configurar el entorno de gestión de la configuración (CM)
Descripción principal

Visión general

Los proyectos de UCM de ClearCase se pueden integrar con ClearQuest, una herramienta de gestión de solicitudes de cambio. Utilizando ClearQuest, puede enviar solicitudes de cambio, ver y modificar solicitudes de cambio existentes, y crear y ejecutar consultas e informes específicos del usuario o del sitio. La integración de UCM y ClearQuest añade posibilidades importantes de gestión de actividades y gestión de proyectos para los proyectos de UCM. Por ejemplo, los registros de ClearQuest se pueden enlazar con políticas y actividades de ClearCase que se pueden establecer para controlar cuándo se desea entregar una actividad de ClearCase, y cuándo se desea cerrar una actividad de ClearQuest.

Esta guía de la herramienta se aplica al ejecutar Microsoft Windows.

  Para obtener más información, consulte la publicación IBM Rational ClearCase Guide to Managing Software Projects.

Pasos de la herramienta

Para establecer la integración de UCM y ClearQuest:

  1. Permitir que un esquema de ClearQuest trabaje con UCM o utilice el esquema habilitado de UCM predefinido
  2. Crear o ampliar una base de datos de solicitudes de cambio para utilizar el esquema
  3. Permitir que el proyecto de UCM trabaje con ClearQuest

1. Permitir que un esquema de ClearQuest trabaje con UCM o utilice el esquema habilitado de UCM predefinido

En los pasos siguientes se indica cómo utilizar esquemas habilitados de UCM predefinidos para establecer una base de datos de solicitudes de cambio para trabajar con UCM. Los esquemas de UCM predefinidos, denominados UnifiedChangeManagement y Enterprise, incluyen el tipo de registro, el campo, el formulario y otras definiciones necesarias para trabajar con un proyecto de UCM.

Para establecer una base de datos de solicitudes de cambio para trabajar con UCM:

  1. Cree un base de datos de usuario que se asocie a uno de los esquemas habilitados de UCM predefinidos.
  2. En ClearQuest Designer, pulse Base de datos > Nueva base de datos para iniciar el asistente para Nueva base de datos.
  3. Complete los pasos del asistente; en el número 4 se le solicita que seleccione un esquema que asociar a la nueva base de datos. Desplace la lista de nombres de esquemas y seleccione el nuevo esquema.
  4. Pulse Finalizar.

  Para obtener más información sobre cómo habilitar un esquema de ClearQuest para utilizarlo con la integración de UCM y ClearQuest, consulte el capítulo que trata sobre el establecimiento de una base de datos de usuario de Rational ClearQuest para UCM en la publicación IBM Rational ClearCase Guide to Managing Software Projects.

2. Crear o ampliar una base de datos de solicitudes de cambio para utilizar el esquema

Gracias a los esquemas de UCM predefinidos, puede utilizar la integración de UCM y ClearQuest inmediatamente, pero es posible que prefiera diseñar un esquema personalizado para hacer un seguimiento de las solicitudes de cambio y las actividades del proyecto. Incluso, es posible que prefiera utilizar otro esquema predefinido. En función de las necesidades del proyecto, existen varios modos de hacerlo.

Antes de continuar, se recomienda que se familiarice con la información que se facilita en el capítulo que trata sobre el establecimiento de una base de datos de usuario de Rational ClearQuest para UCM de la publicación IBM Rational ClearCase Guide to Managing Software Projects. En este apartado del conjunto de documentación de ClearCase se proporciona información detallada sobre los procedimientos que llevar a cabo para la realización de esta tarea.

   Para obtener información detallada sobre este procedimiento, consulte el apartado que trata sobre la planificación para la utilización de la integración de UCM con Rational ClearQuest de la publicación IBM Rational ClearCase Guide to Managing Software Projects.

3. Permitir que el proyecto de UCM trabaje con ClearQuest

  • En el panel situado a la izquierda del Explorador de proyectos de ClearCase, pulse el botón derecho del ratón sobre el proyecto para que se muestre el menú de método abreviado correspondiente. Pulse Propiedades para visualizar la hoja de propiedades.
  • Pulse la pestaña ClearQuest y, a continuación, seleccione el recuadro de selección Proyecto habilitado para ClearQuest. En sistemas Windows, Enlazar a la base de datos de usuario de ClearQuest contiene un nombre. Se muestra el diálogo Depósito de esquemas de Rational ClearQuest con la conexión correspondiente a la base de datos de usuario seleccionada.
  • Seleccione la base de datos de usuario de la lista "Enlazar a esta base de datos de solicitudes de cambio".
  • La primera vez que habilita un proyecto, ClearQuest abre el recuadro de diálogo de Inicio de sesión correspondiente. Entre el nombre de usuario, la contraseña y el nombre de la base de datos a la que va a enlazar el proyecto.
  • Pulse Aceptar.

  Característica avanzada: También puede establecer políticas de desarrollo para proyectos en los que se utiliza la integración de UCM y ClearQuest. Para obtener más información, consulte el apartado que trata sobre las políticas para la integración de UCM con Rational ClearQuest de la publicación IBM Rational ClearCase Guide to Managing Software Projects.
