Resumen

Para aplicar de forma eficaz un proceso de desarrollo de software como el de RUP, es importante comprender primero sus objetivos clave, la importancia de cada uno de ellos y cómo funcionan juntos para ayudar al equipo de desarrollo a crear un producto de calidad que cumpla las necesidades reales de los interesados.
