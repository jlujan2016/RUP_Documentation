Resumen

El modelo de madurez de posibilidades (CMM) de Software Engineering Institute (SEI) proporciona un medidor de rendimiento reconocido para la madurez del proceso de software. El CMM se ha convertido en un vehículo popular para valorar la madurez del proceso de software de una empresa en muchos dominios. En este documento técnico se describe cómo  RUP puede dar soporte a una empresa que está intentando alcanzar los niveles de madurez en el proceso de software CMM 2 (Se puede repetir) y Nivel 3 (Definido).
