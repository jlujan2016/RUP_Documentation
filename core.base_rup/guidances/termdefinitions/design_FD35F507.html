Definición de términos: diseño
Descripción principal
Parte del proceso de desarrollo de software cuyo objetivo principal es decidir la implementación del sistema. Durante el diseño, se toman decisiones estratégicas y tácticas con el fin de cumplir los requisitos funcionales y de calidad de un sistema. Véase: análisis.
