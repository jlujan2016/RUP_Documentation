Content

This plug-in provides formal document templates and examples for most RUP work products.

It includes a set of examples based on an example project, the Wylie College Course Registration System, and stand-alone examples including programming guidelines and use case modeling guidelines for each phase. 

It also provides a number of sample reports. A report extracts information from a tool repository, such as a Rational Rose model or a Rational RequisitePro requirements database.
