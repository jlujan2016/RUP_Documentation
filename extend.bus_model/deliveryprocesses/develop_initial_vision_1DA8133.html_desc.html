Descripción

Este patrón, o conjunto de actividades, agrupa las actividades que se necesitan para desarrollar la visión inicial del proyecto. Como las actividades individuales necesarias para alcanzar el objetivo de desarrollar la visión inicial no son de interés particular en sí mismas, la inclusión de este patrón de alto nivel en los planes de iteración enfatiza el objetivo que se pretende alcanzar, pero deja para el siguiente nivel de detalle los detalles de las actividades que se necesitan para esta acción. Esto permite que un diagrama de actividad, como el flujo de trabajo de iteraciones, muestre las actividades que tienen lugar sin desordenar en exceso el diagrama.
